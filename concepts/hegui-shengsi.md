# 出海合规生死线

出海HR合规中不可触犯的核心底线，一旦触犯可能造成灾难性后果。

## 高危情形

| 情形 | 后果 | 典型市场 |
|------|------|---------|
| 无工作许可雇佣 | 罚款+驱逐+主体被关 | 欧洲、中东 |
| 社保未申报 | 补缴+利息+刑事责任 | 东南亚 |
| 工资拖欠 | 刑事拘留（部分国家）| 中东、东南亚 |
| 数据违规（GDPR）| 最高年营收4%或€2000万 | 欧盟 |
| 虚假劳动合同 | 合同无效+重罚 | 全球 |

- [ ] 社保在入职当月申报
- [ ] 薪资发放符合最低工资标准
- [ ] 数据处理符合GDPR/当地数据法
