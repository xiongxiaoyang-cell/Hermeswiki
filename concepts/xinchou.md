# 薪酬绩效体系

出海企业"选-用-育-留-离"中的"用"与"留"，核心挑战是属地化薪酬体系统一性与灵活性的平衡。

## 核心挑战

- 属地化薪酬 vs 全球一致性：不同国家基线差异巨大
- 外派员工薪酬包设计：Base+补贴+税务补贴+回国假
- 社保合规：各国社保费率差异显著（部分国家无社保）
- 绩效管理：KPI跨文化适配

## 薪酬结构

| 组成 | 说明 |
|------|------|
| Base Salary | 属地市场基准 |
| Housing | 住房补贴（外派专属）|
| Mobility | 调动津贴 |
| Tax Equalization | 税务平衡 |
| Benefits | 属地福利（医疗/养老等）|
