# 飞书回调配置指南

## 飞书应用创建

1. 打开 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用
3. 获取 App ID 和 App Secret

## 权限配置

在飞书开放平台应用后台添加以下权限：
- `im:message` - 接收和发送消息
- `im:chat` - 管理群聊
- `im:message:send_as_bot` - 以机器人身份发送消息

## 事件订阅

在飞书开放平台配置**事件订阅**：
- 填写回调URL（需要公网可访问）
- 添加事件：`im.message.receive_v1`

## OpenClaw 配置

需要在 `~/.openclaw/openclaw.json` 中添加飞书配置：

```json
{
  "accounts": {
    "feishu": {
      "default": {
        "appId": "你的App ID",
        "appSecret": "你的App Secret",
        "connectionMode": "webhook",
        "webhookHost": "https://your-domain.com",
        "webhookPath": "/webhook/feishu",
        "webhookPort": 3000,
        "groupPolicy": "open"
      }
    }
  }
}
```

## 公网访问方案

### 方案1：公网服务器
- 有自己的域名和服务器
- 部署OpenClaw Gateway到公网

### 方案2：内网穿透（推荐开发测试）
- 使用 ngrok：`ngrok http 18789`
- 使用 cloudflare tunnel

### 示例：ngrok 配置
```bash
ngrok http 18789
# 获取公网URL后填入 webhookHost
```
