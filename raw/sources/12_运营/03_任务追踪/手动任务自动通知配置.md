# 手动任务自动通知配置

## 概述

手动触发的子Agent任务，可通过以下方式配置自动通知到群。

---

## 方式一：在task_registry.json中配置

添加一个手动任务，配置notify参数：

```json
{
  "id": "manual_task_example",
  "name": "手动任务示例",
  "type": "subagent",
  "subagent_id": "subagent_ip",
  "message": "执行任务描述",
  "enabled": true,
  "notify": {
    "on_success": true,
    "on_failure": true,
    "on_retry": false,
    "targets": ["群聊ID1", "群聊ID2"]
  }
}
```

---

## 方式二：使用预设的手动任务模板

在 `config/manual_tasks/` 目录下创建任务文件：

```yaml
name: "任务名称"
agent: "subagent_id"
message: "任务描述"
notify_on_success: true
notify_groups:
  - "oc_xxx1"  # 群聊ID
  - "oc_xxx2"
```
