# 国家指南问答 Skill

## 核心能力
- **智能识别**：自动从自然语言中提取国家名称和业务关键词
- **三级数据策略**：本地缓存 → API获取 → 联网搜索（兜底）
- **上下文记忆**：连续对话自动沿用国家上下文，无需重复输入
- **精准回答**：基于官方指南内容，针对劳动法规、薪酬税务、福利休假、合同入职、外企本地化等专业领域提供结构化回答

## SOP 流程

### 第一步：获取国别指南文件

当用户提问时：

1. **提取国家名称**：
   - **情况A**：用户问题中包含国家名称（如"美国的工作签证怎么办"→提取"美国"）→ 使用提取的国家
   - **情况B**：用户问题中**没有**国家名称（如"工作签证怎么办"）→ **使用上下文记忆的国家**（上一次的对话国家）
   - **情况C**：首次对话且无国家名称 → 提示用户指定国家
2. **更新上下文记忆**：将当前使用的国家名称保存到对话上下文中
3. **检查本地缓存**：查看 `output/country-guides/{国家名称}_国别指南.md` 是否已存在
   - 若存在且未过期，直接读取本地文件
   - 若不存在，执行第4步
4. **调用 API 获取**：
   - POST 请求到 `https://flow.anchortech.ai/api/task/countryguide`
   - 请求体：`{"title": "国家名称"}`
5. **解码并保存**：
   - 解码 `content_base64` 内容
   - 保存到 `output/country-guides/{国家名称}_国别指南.md`

### 第二步：根据文件内容回答问题

1. **读取指南文件**：加载对应的 MD 文件内容
2. **分析问题意图**：理解用户具体想问什么（如签证、劳动法、税务等）
3. **检索相关信息**：在指南文件中查找与问题相关的章节
4. **生成回答**：基于指南内容，给出准确、完整的回答

### 第三步：联网搜索（兜底方案）

如果**本地缓存不存在**且**API 调用失败**（返回错误或该国家无指南）：

1. **提取搜索关键词**：从用户问题中提取国家名称 + 核心问题（如"德国 工作签证"）
2. **执行联网搜索**：使用搜索工具获取相关信息
3. **整合搜索结果**：从搜索结果中筛选权威、最新的信息
4. **生成回答**：基于搜索结果回答用户问题，并**注明信息来源为网络搜索**
5. **可选保存**：如搜索结果质量较高，可保存为临时参考文件

> **注意**：联网搜索的回答会标注"基于网络搜索结果"，与官方指南有所区别

从用户问题中自动提取，支持：

- 美国、美利坚合众国
- 日本、英国、德国、法国
- 沙特阿拉伯、新加坡、澳大利亚
- 等...

## API 详情

- **URL**: `https://flow.anchortech.ai/api/task/countryguide`
- **方法**: POST
- **请求体**: `{"title": "国家名称"}`
- **响应**: JSON 格式，包含 `main_text` 字段

## 注意事项

- 确保网络连接正常
- 如果 API 返回错误或该国家无指南，自动启用联网搜索作为兜底方案
- 联网搜索的结果会明确标注来源，与官方指南区分
- **上下文记忆**：连续对话中自动沿用上一次的国家，无需重复输入
- 优先使用本地缓存 > API 获取 > 联网搜索的三级策略
- 如需切换国家，直接在新问题中指定即可
