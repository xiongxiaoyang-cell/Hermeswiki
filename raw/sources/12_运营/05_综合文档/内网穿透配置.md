## 启动命令

```bash
# 启动 localtunnel
lt --port 18789 --subdomain david-openclaw

# 后台运行
lt --port 18789 --subdomain david-openclaw > /tmp/lt.log 2>&1 &
```

---

## 备选方案

### ngrok（需要有效token）
1. 获取 token: https://dashboard.ngrok.com
2. 配置: `ngrok config add-authtoken <TOKEN>`
3. 启动: `ngrok http 18789`

### Cloudflare Tunnel（需要Cloudflare账户）
需要已备案域名

## 使用场景

| 场景 | 推荐方案 |
|------|---------|
| Gitee Webhook | localtunnel / ngrok |
| 飞书回调 | localtunnel / ngrok |
| 临时测试 | localtunnel |
| 长期使用 | ngrok / Cloudflare |

---

## 注意事项

- localtunnel 每次重启后会变化
- 需要长期稳定建议使用 ngrok 付费版或 Cloudflare Tunnel
