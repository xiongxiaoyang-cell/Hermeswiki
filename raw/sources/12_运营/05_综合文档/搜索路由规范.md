## 三、主Agent搜索行为规范

当主Agent需要执行临时搜索时：

```
1. 简单URL抓取（<1分钟）→ 直接 web_fetch
2. 搜索引擎检索（1-5分钟）→ ddg-search skill / web_fetch
3. 深度研究（>30分钟）→ sessions_spawn Echo + Tavily（禁止主Agent自己做）
```

**禁止**：主Agent用M2.7模型做长时间的网页遍历和内容总结（消耗大、效果差）
