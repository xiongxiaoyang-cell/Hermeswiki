# 市场预测准确度持续改进规则

> **目的**：通过数据驱动的信号校准、跨信号共识机制和预测结果追踪，持续提升A股/港股次日走势预测准确率。
> **创建时间**：2026-03-27
> **状态**：v1.0 — 运行中，持续迭代

---

## 一、预测验证机制

### 1.1 验证时间线

```
T日 07:00-10:30    →  发布晨报，预测T日走势，保存到 prediction_history.json
T日 16:05         →  A股收盘，market_prediction_validator.py 执行验证
                      - 获取T日实际上证收盘价
                      - 对比T日预测结论 vs T日实际涨跌
                      - 更新 prediction_validation.json
T+1日 07:XX       →  次日晨报自动展示：昨日预测准确/偏差 + 历史准确率
```

### 1.2 预测准确率统计

- **文件**：`memory/prediction_validation.json`
- **统计维度**：
  - 方向准确率（涨/平/跌三分类）
  - 各信号的预测有效率（哪些信号预测更准）
  - 跨信号共识 vs 单信号预测的准确率对比

### 1.3 准确率目标

| 阶段 | 目标准确率 |
|------|-----------|
| v1.0（当前）| 方向准确 ≥ 60% |
| v2.0 | 方向准确 ≥ 70% |
| v3.0 | 方向准确 ≥ 75% + 幅度预测 |

## 二、信号有效性评级（v1.0 经验值）

基于A股特性，持续记录各信号的预测有效性：

### A级 — 强预测信号

| 信号 | 权重 | 判定规则 | 已知偏差 |
|------|------|---------|---------|
| **A50期货** | ±2分 | 涨≥0.3% → +2，跌≤-0.3% → -2 | A50在A股开盘前30分钟最准确 |
| **北向资金** | ±1分 | 净买入>10亿→+1，净卖出>10亿→-1 | 盘中数据不完整，以盘后为准 |

### B级 — 中等预测信号

| 信号 | 权重 | 判定规则 | 已知偏差 |
|------|------|---------|---------|
| **上证开盘强度** | ±1分 | 沪指+创业板同时涨/跌超过阈值 | 开盘半小时内波动大，需等待 |
| **港股恒指** | ±1分 | 恒指涨跌>1.5%时触发 | 港股对A股有领先性但非绝对 |
| **韩国KOSPI** | ±1分 | KOSPI涨跌>1.5%时触发 | 韩国市场情绪有区域联动性 |
| **USD/CNY** | ±1分 | 涨跌>0.5%时触发 | 阈值不宜过严，避免噪音 |

### C级 — 参考信号（不直接打分）

| 信号 | 作用 | 已知偏差 |
|------|------|---------|
| 美股昨夜均值 | ±1分 | 美股只影响A股开盘，后续A股独立性强；除非极端波动（>2%）|
| 黄金/原油联动 | 综合打分 | 需结合两者方向，单独看黄金/原油意义有限 |
| VIX恐慌指数 | 中性参考 | VIX>25不代表A股跌，只是市场不确定性信号 |

### D级 — 暂不可用信号

| 信号 | 状态 | 解决方案 |
|------|------|---------|
| A50夜盘（腾讯接口） | ❌ 失效 | ✅ 已切换至 akshare futures_global_spot_em |
| 财经新闻关键词搜索 | ❌ 网络不通 | ✅ 已切换至同花顺实时新闻API |

## 三、跨信号共识机制

**核心原则**：当多个独立信号指向同一方向时，预测可信度大幅提升。

### 3.1 共识等级

| 共识等级 | 触发条件 | 加成规则 |
|---------|---------|---------|
| 强共识 | ≥4个信号同向 | score × 1.5（四舍五入），倾向该方向 |
| 中共识 | 3个信号同向 | score × 1.2，倾向该方向 |
| 弱共识 | 2个信号同向 | 无加成，正常计分 |
| 分歧 | 3+个信号对立 | 依赖A50单一信号，削弱整体可信度 |

### 3.2 共识信号分类

**看多信号**：A50涨、北向净买入、港股涨、USD/CNY跌、黄金跌
**看空信号**：A50跌、北向净卖出、港股跌、USD/CNY涨、黄金+原油双涨

## 四、评分阈值动态调整规则

**当前固定阈值（待校准）**：

```
score ≥ +3  → 📈 乐观
score = +2  → 📊 谨慎乐观
score = +1  → ↔️ 中性偏多
score =  0  → ↔️ 中性
score = -1  → ↔️ 中性偏空
score = -2  → 📉 谨慎悲观
score ≤ -3  → 🔻 悲观
```

### 调整触发条件

当历史准确率出现以下情况时，自动触发阈值复盘：

| 触发条件 | 动作 |
|---------|------|
| 连续3次"乐观"预测实际均下跌 | 重新评估看多信号权重 |
| 连续3次"悲观"预测实际均上涨 | 重新评估看空信号权重 |
| 连续5次中性预测中4次方向正确 | 考虑收紧中性区间阈值 |
| 任意信号连续5次预测与实际不符 | 降级该信号权重或改为参考信号 |

## 五、每日复盘流程（Heartbeat时执行）

每次heartbeat检查以下内容：

1. **读取 `prediction_validation.json`**
   - 昨日预测是否准确？
   - 哪些信号预测有效？哪些失灵？

2. **如果连续2次预测错误**
   - 分析错误原因（哪个信号误导了？）
   - 在 `skills/market-predictor/rules_v1.md` 中记录偏差案例

3. **如果某信号连续3次失灵**
   - 自动降级该信号权重
   - 在报告中标注"⚠️ 该信号近期预测失灵"

4. **每月末统计**
   - 生成月度准确率报告
   - 校准各信号权重
   - 更新本规则的 `v1.x` 版本号

## 六、已知市场规律（待量化验证）

以下规律来自A股历史经验，尚未进入评分模型，**仅供辅助参考**：

1. **A股开盘"三点定势"规律**：开盘后30分钟内，上证走势基本确定当日大方向
2. **节假日前效应**：节假日前1个交易日，A股通常缩量横盘
3. **美股→A股传导时滞**：美股涨跌 >2%时，对A股影响较大；<1%时A股常走出独立行情
4. **北向资金"反向指标"现象**：北向某日大幅净卖出后，往往次日A股反而反弹（需更多数据验证）
5. **黄金/股市"跷跷板"**：黄金持续大涨（>2%/日）通常对应股市资金流出压力
