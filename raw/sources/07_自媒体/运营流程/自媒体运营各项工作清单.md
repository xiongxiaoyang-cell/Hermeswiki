## 二、原则落实方案

### 核心理念：轻松好奇，探索式，不追求数量

| 原则 | 落地措施 |
|------|----------|
| **不追求数量** | 去掉固定更新频率，按灵感触发 |
| **轻松投入** | 每次内容控制在15分钟内完成 |
| **好奇驱动** | 选题来源改为"今天想分享什么" |
| **形式多样** | 不限制形式：短图文/口播/VLOG/纯文字都可以 |
| **有成就感** | 每完成一条就是成功，不看数据焦虑 |

## 三、执行细则

### 1. 热点追踪（轻量版）

| 做法 | 说明 |
|------|------|
| 有看到才追踪 | 不主动定时搜索 |
| 15分钟Limit | 超过就停下 |
| 只记录感兴趣 | 不感兴趣的热点不强迫自己做 |

### 2. 选题操作

| 做法 | 说明 |
|------|------|
| 灵感池 | 只记录，不强制完成 |
| 想做才做 | 有状态/灵感时从池子里挑 |
| 不强求 | 没状态就不做 |

### 3. 内容生产

| 做法 | 说明 |
|------|------|
| 时间弹性 | 有空就做，没空不做 |
| 形式随意 | 什么形式都行 |
| 质量优先 | 不想发的不发 |
| 短平快 | 不追求完美，接受粗糙 |

### 4. 发布节奏

| 做法 | 说明 |
|------|------|
| 无固定节奏 | 不设日更/周更目标 |
| 想发才发 | 有内容想分享就发 |
| 不比较 | 不和别的账号比 |
