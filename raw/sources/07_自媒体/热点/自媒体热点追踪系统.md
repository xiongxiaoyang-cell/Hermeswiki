## 二、选题追踪频率

| 时间 | 内容 |
|------|------|
| **每周日下午** | 选题会，给出下周3条内容建议 |
| **每天早上（8-9点）** | 全渠道热点：抖音+视频号+小红书+知乎+百度 |
| **每天中午（12-13点）** | 热点更新：微博+百度+抖音午间热点 |
| **每天晚上（20-21点）** | 热点复盘：当日热点+次日选题灵感 |
| 发布前1天 | 最终热点确认，确保时效性 |

---

### 每日热点追踪清单（全渠道）

- [ ] 抖音热榜（douyin.com/hot）
- [ ] 视频号/微信热榜
- [ ] 小红书搜索发现
- [ ] 知乎热榜
- [ ] 百度热搜
- [ ] 微博热搜

## 四、各平台调性对比

| 平台 | 内容风格 | 时长 | 标题特点 |
|------|----------|------|----------|
| 抖音 | 快、燃、猎奇 | 15-60秒 | 震惊体、数字冲击 |
| 视频号 | 务实、民生 | 30-90秒 | 干货感、接地气 |
| 小红书 | 精致、实用 | 图文/1-3分钟 | 种草风、清单体 |
| 知乎 | 深度、专业 | 1000字+ | 问答体、分析型 |

## 五、执行流程

1. **每日热点追踪**（早中晚3次全渠道） → 2. **每周日下午选题会** → 3. **脚本创作** → 4. **录制发布**
