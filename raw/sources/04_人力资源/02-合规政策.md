# 模块2：政策层 - 全球用工合规政策

## 2.1 四大政策领域

| 领域 | 核心内容 |
|------|----------|
| **用工政策** | 劳动合同、试用期、工时、假期、解雇规则、反歧视法 |
| **薪税政策** | 最低工资、社保费率、个税计算、税务申报 |
| **移民政策** | 工作许可、签证类型、居留许可、签证续签 |
| **数据合规** | 员工信息收集、存储、使用、传输（GDPR等）|

## 2.2 合规管控流程

```
招聘阶段 → 入职阶段 → 在职阶段 → 离职阶段
   ↓          ↓          ↓          ↓
  JD合规    合同签署   薪税申报   离职结算
 渠道合规    入职手续   社保缴纳   证明开具
 背景调查    签证办理   绩效评估   竞业管理
```

## 2.3 政策变动应对

| 动作 | 说明 |
|------|------|
| 预警机制 | 对接属地律所/税务/移民局，月度更新清单 |
| 解读团队 | 明确政策变动对HR工作的影响 |
| 流程调整 | 及时调整HR流程、模板、方案 |
| 合规自查 | 政策变动后开展专项排查 |
