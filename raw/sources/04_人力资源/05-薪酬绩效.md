# 模块5：薪酬层 - 薪酬绩效与成本核算

## 5.1 薪酬方案设计

| 方案 | 适用场景 | 设计逻辑 |
|------|----------|----------|
| **派出国平衡法** | 短期外派（≤3年） | 派出国薪酬+外派激励+生活津贴 |
| **Local+法** | 长期全球化 | 东道国薪酬+专属福利津贴 |

## 5.2 核心成本构成

| 成本项 | 说明 |
|--------|------|
| 基本工资 | 月度/年度固定薪酬 |
| 绩效奖金 | 与业绩挂钩 |
| 法定福利 | 社保、养老金等 |
| 额外福利 | 住房、子女教育、商业保险 |
| 隐性成本 | 签证费用、搬迁费、税收成本 |

## 5.3 薪酬测算公式

```
总包成本 = 基本工资 × 12 + 绩效奖金 + 法定福利 + 额外福利 + 隐性成本

时薪成本 = 总包成本 ÷ (标准工时 × 52周)
```
