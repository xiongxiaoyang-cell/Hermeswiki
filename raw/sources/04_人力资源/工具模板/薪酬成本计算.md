# 知识卡片：薪酬成本计算

> 一页速查 / 持续更新

---

## 薪酬总包构成

```
总包成本 = 基本工资 + 绩效奖金 + 法定福利 + 额外福利 + 隐性成本
```

---

## 成本明细

| 成本项 | 说明 | 占比参考 |
|--------|------|----------|
| 基本工资 | 月薪×12或年薪 | 60-70% |
| 绩效奖金 | 与业绩挂钩 | 5-15% |
| 法定福利 | 社保、养老金 | 15-25% |
| 额外福利 | 住房、教育、保险 | 5-10% |
| 隐性成本 | 签证、搬迁、税收 | 3-5% |

---

## 各国成本系数

| 国家 | 成本系数 | 说明 |
|------|----------|------|
| 日本 | 1.4-1.5 | 社保约41% |
| 澳洲 | 1.15-1.2 | 养老金12% |
| 德国 | 1.4-1.5 | 社保约40% |
| 英国 | 1.2-1.3 | NIC约13.8% |
| 美国 | 1.3-1.4 | 社保+医保 |

> 成本系数 = 总成本 ÷ 基本工资

---

## 计算示例

**日本员工，月薪30万日元**
```
基本工资：30万 × 12 = 360万
社保（41.3%）：30万 × 41.3% × 12 = 148.7万
奖金（2个月）：30万 × 2 = 60万
--------------------------------
年度总成本：约 568.7万日元
成本系数：1.58
```

---

## 时薪计算

```
时薪成本 = 年度总成本 ÷ (标准工时 × 52周)
         = 568.7万 ÷ (40 × 52)
         = 2,734 日元/时
```
