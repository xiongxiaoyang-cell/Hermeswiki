## 出海三阶段流程

```
┌─────────────────────────────────────────────────────────┐
│                    试水期 (0-10人)                       │
├─────────────────────────────────────────────────────────┤
│  调研 → 选址 → EOR签约 → 招聘 → 合规落地                │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    增长期 (10-50人)                      │
├─────────────────────────────────────────────────────────┤
│  主体设立 → 自雇佣 → 规模化 → 薪酬BPO                    │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    全球化 (50人+)                        │
├─────────────────────────────────────────────────────────┤
│  属地化 → 领导力 → 总部协同 → 全球化治理                │
└─────────────────────────────────────────────────────────┘
```

---

## 关键节点

| 阶段 | 核心动作 | 周期 |
|------|----------|------|
| 试水期 | EOR签约、第一人招聘 | 1-2个月 |
| 增长期 | 海外主体、规模化招聘 | 3-6个月 |
| 全球化 | 属地管理、全球HR体系 | 6-12个月 |

---

## 落地10步法

1. **定战略** - 明确出海目标与阶段
2. **查政策** - 梳理目标国合规要求
3. **选组织** - 确定用工模式
4. **建标准** - 人才标准与招聘方案
5. **算成本** - 薪酬总包与测算
6. **搭体系** - 全周期合规流程
7. **套工具** - 模板与系统
8. **对案例** - 参考行业标杆
9. **配服务** - 对接落地服务商
10. **复盘迭代** - 持续优化
