## 五、海外业务发展阶段与HR管理

### 1. 市场探索期
| 项目 | 内容 |
|------|------|
| 模式 | EOR/PEO服务商 |
| 团队 | 1-10人 |
| 薪酬 | 简单结构，吸引力优先 |
| 优势 | 快速进入，规避风险 |

### 2. 市场拓展期
| 项目 | 内容 |
|------|------|
| 模式 | EOR+自主管理混合 |
| 团队 | 10-50人 |
| 核心岗位 | 自主雇佣 |
| 薪酬 | 体系化框架设计 |

### 3. 成熟运营期
| 项目 | 内容 |
|------|------|
| 模式 | 自主管理为主 |
| 团队 | 50人以上 |
| 薪酬 | 精细化、个性化 |
| 重点 | 雇主品牌建设 |

### 4. 全球化运营期
| 项目 | 内容 |
|------|------|
| 模式 | 区域总部+共享服务中心 |
| 团队 | 多国分布 |
| 薪酬 | 全球统一框架+本地调整 |
| 工具 | HRIS系统、大数据分析 |

## 六、出海服务商使用

### 使用场景
1. 无当地实体阶段（EOR快速合规）
2. 特定国家合规需求
3. 临时性/辅助性岗位

### 服务商类型
| 类型 | 代表 |
|------|------|
| 央企背景 | 中国联通（香港） |
| 全球EOR | Safeguard Global、Deel |
| 万领钧Knit | Payroll+EOR |
| 本地派遣 | 劳务派遣公司 |

### 服务内容
- **EOR服务**：名义雇主，承担法定责任
- **Payroll服务**：薪酬核算、税务申报
- **劳务派遣**：临时性/辅助性岗位

## 七、决策考量因素

| 因素 | 说明 |
|------|------|
| 合规风险 | 降低当地法规风险 |
| 时间效率 | 快速进入市场 |
| 成本控制 | 小规模团队更划算 |
| 管理便捷 | 专注核心业务 |

---

## 八、管理建议

1. **分层管理**：集团宏观 + 驻港微观
2. **本地化战略**：中方外派 + 属地招聘
3. **合规前置**：先合规再运营
4. **服务商组合**：EOR起步 → 自主管理
5. **人才培养**：建立国际化人才梯队
