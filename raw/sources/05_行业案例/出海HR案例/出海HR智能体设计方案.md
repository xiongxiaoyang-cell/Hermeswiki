# 出海HR智能体设计方案

## 摘要

基于《中企出海人力资源管理痛点与解决方案研究报告》的八大案例深度分析，本方案设计面向出海HR从业者的AI智能体产品——**「扬帆HR」**。该智能体以“合规先行、智能决策、全流程覆盖”为核心理念，覆盖合规政策、招聘与雇佣、薪酬与发薪、员工关系与文化融合、组织架构与决策流程、供应商选择与管理、成本控制、信息化系统八大维度，为中企出海HR管理提供一站式AI解决方案。

---

## 一、智能体定位

### 1.1 产品命名与定位

**产品名称**：扬帆HR（Yangfan HR Agent）

**产品定位**：面向中企出海HR从业者的AI智能助手，基于大语言模型与专业知识库，提供出海合规政策解读、多国薪资核算、跨文化员工关系管理、供应商智能匹配等全流程服务。

**核心价值主张**：
- **合规护航**：实时更新50+国家劳动法、税法、社保政策，合规风险秒级预警
- **智能决策**：基于案例库与行业数据，提供供应商比选、成本测算、组织架构设计等决策支持
- **效率倍增**：自动化薪资核算、入职流程、报表生成，HR工作效率提升300%

### 1.2 目标用户画像

| 维度 | 描述 |
|------|------|
| **企业类型** | 中大型民营企业、上市公司、国企出海业务部门 |
| **出海阶段** | 布局期（3-10个国家）至扩张期（10+国家） |
| **行业分布** | 科技/互联网、新能源/制造、金融、生命科学 |
| **企业规模** | 海外员工50-500人，年营收海外占比20%以上 |
| **HR角色** | 海外HR负责人、全球HR负责人、HRSSC负责人、HRBP |
| **核心痛点** | 合规政策不清、多国管理复杂、供应商选择困难、数据分散 |

**典型用户画像**：
- **王婷，某新能源车企海外HR负责人**：公司正在拓展沙特、印尼市场，需要同时管理3个国家的员工，对各国社保缴纳、劳动合同条款不熟悉，希望有智能工具帮她快速了解合规要求并找到靠谱的EOR供应商。
- **李明，某金融科技公司HRD**：公司刚进入英国市场，HR团队没有出海经验，需要系统性地了解英国用工合规、发薪、签证等全流程政策。

## 二、功能维度设计

### 2.1 【合规政策】模块

**核心功能描述**：
智能合规政策助手，基于覆盖50+国家的劳动法、税法、社保政策数据库，提供实时政策查询、条款解读、合规风险预警，支持劳动合同模板生成与审查。

**具体使用场景**：
```
场景：某新能源车企HR陈思远，公司计划进入沙特市场
- 登录扬帆HR，输入“沙特劳动法试用期规定”
- 系统秒级返回沙特《劳动法》关于试用期的具体条款
- 自动标注与中国法律的差异点
- 推送相关合规风险提示（如试用期延长条件、解雇补偿计算）
- 提供沙特标准劳动合同模板下载
```

**输入/输出**：
- **输入**：国家/地区名称、政策类型（劳动法/税法/社保/签证）、具体问题
- **输出**：政策原文摘要、关键条款对比、合规建议、相关模板

**典型用户问题示例**：
- “英国雇主必须缴纳的National Insurance Contributions税率是多少？”
- “德国解雇员工需要提前多久通知？”
- “印尼员工年度带薪休假天数是多少？”
- “沙特阿拉伯籍员工与外籍员工在社保方面有什么区别？”

**知识库需整合内容**：
- 50+国家《劳动法》原文与中文翻译
- 各国社保/税务法规原文
- 各国签证政策文件
- 劳动合同条款中西方差异分析
- 合规风险案例库（处罚案例、纠纷案例）

### 2.2 【招聘与雇佣】模块

**核心功能描述**：
智能招聘与雇佣助手，整合多国人才市场数据，提供薪资基准查询、招聘周期评估、猎头供应商匹配，支持全流程入职办理指引。

**具体使用场景**：
```
场景：某AI SaaS公司HRBP刘芳，计划拓展巴西市场
- 输入“巴西圣保罗招Java工程师，5年经验，薪资范围？”
- 系统返回：圣保罗Java工程师平均薪资（来源：Glassdoor/LinkedIn数据）
- 提供招聘周期预估：6-10周
- 推送在巴西有Java工程师交付能力的猎头供应商名单
- 如客户已有候选人，提供巴西offer谈判注意事项
```

**输入/输出**：
- **输入**：目标国家、城市、岗位类型、工作年限、资质要求
- **输出**：薪资基准报告、招聘周期评估、供应商推荐、入职流程清单

**典型用户问题示例**：
- “越南胡志明市，软件测试工程师年薪多少钱？”
- “印尼本地候选人期望的薪资一般是多少？”
- “我们要在马来西亚招聘10个销售，招聘周期多久？”
- “哪些猎头供应商在泰国曼谷有科技人才交付能力？”

**知识库需整合内容**：
- 各国薪资基准数据库（按岗位/城市/经验）
- 招聘周期参考数据
- 猎头供应商能力评估档案
- 各国雇主品牌建设案例
- 入职流程模板（各国）

### 2.3 【薪酬与发薪】模块

**核心功能描述**：
智能薪酬与发薪助手，支持多国薪资核算，提供多币种换算、税务计算、社保计算、年度人力成本测算，自动生成发薪指南。

**具体使用场景**：
```
场景：某金融科技公司HR张明，英国团队10人，需要算薪
- 输入“英国员工月薪£5000，年终奖£10000，请计算年度人力成本”
- 系统返回：
  - 月薪：£5000 × 12 = £60000
  - 年终奖：£10000
  - 雇主National Insurance：约£5200
  - 雇主养老金缴费：约£2700
  - 年度人力成本总计：约£73900
- 推送英国发薪时间节点、报税节点提醒
```

**输入/输出**：
- **输入**：员工月薪/年薪、国家、货币、发放周期、汇率
- **输出**：薪资核算明细、税费/社保计算结果、年度人力成本报告

**典型用户问题示例**：
- “日本员工月薪35万日元，个人所得税如何计算？”
- “德国员工月薪€6000，雇主需要承担哪些社保费用？”
- “外派到沙特的员工，驻外津贴如何设计才合规免税？”
- “多币种发薪，汇率波动风险如何对冲？”

**知识库需整合内容**：
- 各国薪资税制计算规则
- 各国社保/公积金计算规则
- 多币种实时汇率数据源
- 外派人员薪资结构设计指南
- 年终奖税务规划案例

### 2.5 【组织架构与决策流程】模块

**核心功能描述**：
智能组织架构与决策流程助手，提供全球HR组织模式设计、多国管控权限划分、决策流程优化建议。

**具体使用场景**：
```
场景：某消费电子公司HRD赵总，公司从3国扩张到10国
- 输入“公司海外业务从3国扩张到10国，HR组织架构如何设计？”
- 系统返回：
  - 阶段判断：布局期→扩张期
  - 建议模式：建立区域HR中心（如亚太/欧洲/中东）
  - 总部职责：政策制定、核心岗位招聘、供应商管理

  - 决策权限划分：按事项类型（入职/离职/薪酬调整）设定审批层级
- 推送全球HR管控框架模板
```

**输入/输出**：
- **输入**：国家数量、业务复杂度、现有组织架构
- **输出**：组织架构设计方案、决策权限矩阵、流程优化建议

**典型用户问题示例**：
- “海外子公司HR负责人向谁汇报比较合理？”

- “总部HR和海外本地HR的职责如何划分？”

**知识库需整合内容**：
- 全球HR组织模式案例库

**核心功能描述**：
智能供应商选择与管理助手，整合EOR/猎头/律所等供应商数据库，提供供应商能力评估、智能匹配推荐、合同条款审查、服务质量监控。

### 工作流2：多国发薪月度处理（Monthly Payroll Workflow）

### 工作流3：员工全生命周期管理（Employee Lifecycle Workflow）

**触发条件**：
- 入职触发：HR输入“新员工入职，帮我处理入职手续”
- 离职触发：HR输入“XX员工要离职，需要办理什么手续”
- 转岗触发：HR输入“XX员工要转调海外，薪资怎么调整”

**步骤分解（入职场景）**：
```
Step 1: 入职信息登记
   → 收集：员工姓名、国家、岗位、入职日期、薪资
   → 判断：入职国家

Step 2: 入职清单生成
   → 触发 Skill：合规政策解读
   → 输出：该国入职必要手续清单

Step 3: 劳动合同生成
   → 触发 Skill：合同条款审查
   → 输出：该国标准劳动合同
   → 供HR下载使用

Step 4: 薪资方案确认
   → 触发 Skill：全球薪资核算
   → 输入：约定薪资
   → 输出：完整薪资方案（含税前、税后、社保、年度成本）

Step 5: 跨文化准备
   → 触发 Skill：员工关系与文化融合
   → 输出：该国入职文化指南

Step 6: 办理跟踪
   → 生成入职办理跟踪表
   → 设置重要节点提醒
```

**涉及的Skill组合**：
- 合规政策解读（主力）
- 全球薪资核算（辅助）
- 合同条款审查（辅助）
- 员工关系与文化融合（辅助）

## 七、交付总结

### 核心价值总结

| 维度 | 痛点 | 扬帆HR解决方案 |
|------|------|----------------|
| **合规政策** | 100%企业面临合规挑战，对政策“0基础” | 50+国家政策库+实时更新+AI条款解读 |
| **招聘与雇佣** | 75%企业面临招聘困难，不了解当地市场 | 全球薪资基准+人才市场分析+供应商匹配 |
| **薪酬与发薪** | 多国薪资核算复杂，隐藏成本难识别 | 多国自动算薪+税费/社保一键计算+年度成本报告 |
| **员工关系** | 跨文化沟通障碍，外派员工适应困难 | 跨文化沟通指南+外派员工培训+本地化建议 |
| **组织架构** | 集团管控与本地授权平衡困难 | 全球HR组织模式库+决策权限矩阵+流程设计 |
| **供应商管理** | 62.5%企业受困于供应商选择 | 供应商能力数据库+智能匹配+合同审查 |
| **成本控制** | 合规/招聘成本超预期，预算偏差大 | 多国成本测算+隐藏成本识别+优化建议 |
| **信息化系统** | 系统整合难，数据孤岛严重 | HRIS选型指南+系统集成方案+GDPR合规 |

### 关键差异化

1. **AI原生架构**：不同于传统EOR服务商，7×24小时即时服务，秒级响应
2. **知识图谱驱动**：整合政策库+案例库+模板库+行业数据，深度推理能力
3. **决策智能**：基于历史案例提供供应商比选、成本测算等决策建议
4. **行业垂直**：针对科技/制造/金融/生命科学提供行业专项解决方案
