## 项目背景

名创优品正处于全球化扩张阶段，现有海外各国独立系统无法满足集团统一管控需求。本次项目旨在：
- 建立全球统一HR管理平台
- 实现数据回流中国（跨境合规要求）
- 满足各国差异化合规要求

## 核心需求

### 1. 功能范围（33个模块）
- 组织管理（批量组织管理、组织架构图等）
- 招聘管理（需求管理、OFFER审批）
- 人事事件（入职、转正、异动、晋降任免、外派）
- 合同管理（续签、转签）
- 员工信息管理（档案管理）
- 薪酬管理（调薪、薪资审批、薪资发放审批，不含核算）
- 报表（花名册、异动明细、调薪记录、工资明细等）
- 离职办理
- 员工自助、管理者自助

### 3. 系统集成需求

各国需集成的本地系统：飞书、OA、ADP、IAM、Ento、Brightsafe、Rest、Humanforce、Sunfish-GreatDay HR、People Quest等。

### 4. 合规需求
- 劳动法合规清单
- 数据存储与传输合规性
- 审计追踪与报告
- 多语言支持（中文、英语、德语、法语、泰语、意大利语、波兰语）
- 多时区适配

### 5. 数据合规要点（客户私聊补充）
- **集团要求数据回流中国** —— 跨境数据传输合规是核心挑战
- 要用**一套系统统一覆盖全球**，非各国独立系统
- 核心难点：总部统一管控 vs 各国差异化合规要求的"拉齐"

## 用友解决方案

### 三级合规管控体系
- **总部**：规则制定、战略规划、风险总控
- **区域**：合规审核、风险预警、政策校准
- **属地**：流程执行、员工关系、数据反馈

### 全生命周期闭环
1. 准入与入职管理 — Offer风控、合同合规
2. 在职日常管理 — 考勤、职级、人才发展、合规管控
3. 风险与成本管控 — 试用期、优化管理
4. 离职与关系解除 — 标准SOP、红线禁令

### 配套服务
- 用友BIP系统支撑
- 全球薪资Payroll BPO
- 法规咨询、全球雇员雇佣/解雇建议
- 华为/奇瑞案例参考
