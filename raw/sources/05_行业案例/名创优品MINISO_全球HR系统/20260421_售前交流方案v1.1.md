## 方案定位

用友BIP + Payroll BPO服务，支撑名创优品全球HR系统建设。

## 三级合规管控体系

```
总部（大后方）
    ↓
区域合规中心（NCE）
    ↓
属地运营（各国HR团队）
```

| 层级 | 职责 |
|------|------|
| **总部** | 集团政策制定、数据汇总、战略决策 |
| **区域合规中心（NCE）** | 区域标准、跨国家协调、合规监督 |
| **属地运营** | 本地执行、劳动法合规、当地HR运营 |

## 全生命周期闭环

```
入职 → 在职 → 风险管控 → 离职
  ↑_________________________↓
```

| 阶段 | 核心内容 |
|------|---------|
| **入职** | 招聘、入职手续、合同签订、系统开通 |
| **在职** | 薪酬、考勤、绩效、调动 |
| **风险管控** | 合规监控、劳动法预警、风险预警 |
| **离职** | 离职手续、薪资结算、知识转移 |

## 系统支撑

- **用友BIP**：核心HR系统平台
- **Payroll BPO**：薪资外包服务
