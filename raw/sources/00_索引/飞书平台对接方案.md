# 飞书平台对接方案

> 本方案指导如何连接飞书Wiki和云盘，实现知识库的双向同步

## 二、接入前准备

### 2.1 必要条件

| 条件 | 说明 |
|------|------|
| 飞书企业账号 | 需要管理员权限 |
| 飞书开发者后台 | open.feishu.cn/app |

| App Secret | 应用访问密钥 |

### 2.2 权限规划

**必须开通的权限范围：**

| 权限名称 | 权限说明 | 用途 |
|----------|----------|------|

| wiki | 知识库管理 | 读写Wiki节点 |
| bitable | 多维表格 | 业务数据管理 |

## 三、应用创建流程

### 第一步：创建自建应用

1. 访问 [飞书开发者后台](https://open.feishu.cn/app)
2. 点击 **「创建企业自建应用」**
3. 填写应用信息：
   - 应用名称：`知识库同步助手`
   - 应用描述：`用于知识库与飞书Wiki/云盘的同步`
   - 应用图标：可选
4. 点击 **「创建」** 后获取 `App ID` 和 `App Secret`

### 第二步：添加应用能力

在应用详情页 → **「添加应用能力」**：

| 能力 | 说明 |
|------|------|
| 机器人 | 用于接收消息和主动推送 |
| 网页应用 | 可选，用于管理界面 |
| 小程序 | 可选 |

### 第三步：配置权限

在应用详情页 → **「权限管理」** → 开通以下权限：

```
# 文档权限
docx:document 或 docx:document:readonly
docx:document:create

# 云空间权限
drive:drive 或 drive:drive:readonly  
drive:file 或 drive:file:readonly
drive:folder 或 drive:folder:readonly

# 知识库权限
wiki:wiki 或 wiki:wiki:readonly
wiki:space 或 wiki:space:readonly
wiki:node 或 wiki:node:readonly

# 多维表格权限
bitable:app 或 bitable:app:readonly
```

### 第四步：发布应用

1. 在 **「版本管理与发布」** 创建版本
2. 填写版本信息
3. 选择可用范围（建议先选择部分成员测试）
4. 提交审核并发布

### 第五步：获取Access Token

使用 App ID 和 App Secret 获取访问凭证：

```bash
curl -X POST "https://open.feishu.cn/open-apis/auth/v3/tenant_access_token/internal" \
  -H "Content-Type: application/json" \
  -d '{"app_id":"cli_xxxxxxxxx","app_secret":"xxxxxxxxxxxxxxxx"}'
```

## 四、API对接实现

### 4.1 知识库API（Wiki）

**核心接口：**

| 接口 | 方法 | 说明 |
|------|------|------|
| 获取知识空间列表 | GET /wiki/v2/spaces | 列出所有Wiki空间 |
| 获取节点信息 | GET /wiki/v2/spaces/{space_id}/nodes | 获取节点列表 |
| 获取节点内容 | GET /wiki/v2/spaces/{space_id}/nodes/{node_token} | 读取节点详情 |
| 创建节点 | POST /wiki/v2/spaces/{space_id}/nodes | 创建新节点 |
| 更新节点内容 | PUT /wiki/v2/spaces/{space_id}/nodes/{node_token} | 更新节点 |

### 4.2 云空间API（云盘）

**核心接口：**

| 接口 | 方法 | 说明 |
|------|------|------|
| 获取根目录 | GET /drive/v1/files | 获取根目录文件列表 |
| 获取文件夹内容 | GET /drive/v1/files/{file_token}/children | 获取文件夹子项 |
| 上传文件 | POST /drive/v1/files/upload | 上传文件到云盘 |
| 下载文件 | GET /drive/v1/files/{file_token}/download | 下载文件 |

## 五、OpenClaw集成方案

### 5.1 配置飞书凭证

在 OpenClaw 配置文件中添加：

```json
{
  "feishu": {
    "app_id": "cli_xxxxxxxxx",
    "app_secret": "xxxxxxxxxxxxxxxx",
    "tenant_id": "your_tenant_id"
  }
}
```

### 5.2 权限授予方式

**方式一：添加应用为协作者（推荐）**
1. 打开Wiki页面
2. 点击右上角「...」→「更多」→「添加应用」
3. 搜索并添加知识库同步助手

**方式二：添加应用为知识库管理员**
1. Wiki空间设置 → 成员 → 添加管理员
2. 搜索应用名称添加
