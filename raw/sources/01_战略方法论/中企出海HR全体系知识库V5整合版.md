# 模块1：战略层 - 顶层设计与目标拆解

## 1.1 战略核心定位

> **以"合规为底线、人才为核心、效率为目标、协同为支撑"，搭建适配企业出海阶段的HR体系**

### 四大核心目标
- **合规落地**：用工合规率100%、薪税申报零逾期、签证合规率100%
- **人才适配**：核心岗位招聘到岗率≥90%、试用期留存率≥85%
- **成本可控**：海外用工成本偏差≤5%、人均管理成本逐年下降10%
- **组织协同**：跨文化协作冲突发生率≤5%、员工敬业度≥85%

---

## 1.2 出海三阶段模型

| 阶段 | 核心特征 | 人员规模 | HR战略重点 | 推荐用工模式 |
|------|----------|----------|------------|--------------|
| **试水期** | 无海外主体、业务试错 | ≤10人 | 快速合规落地 | EOR服务 |
| **增长期** | 多国家复制、批量招人 | 10-50人 | 标准化、可复制 | EOR+自雇佣 |
| **全球化** | 属地化深耕、总部协同 | ≥50人 | 全球化治理 | 自雇佣为主 |

---

## 1.3 "选-用-育-留-离"全链路

| 环节 | 核心内容 |
|------|----------|
| 选 | 全球人才获取、外派管理、招聘渠道 |
| 用 | 薪酬绩效、用工合规、绩效管理 |
| 育 | 人才培养、外派培训、领导力发展 |
| 留 | 员工关系、跨文化沟通、激励机制 |
| 离 | 合规离职、离职面谈、知识转移 |

---

## 1.4 合规风控四原则

1. **属地优先** - 严格遵循目标国法律法规，不套用国内经验
2. **流程留痕** - 所有操作留存书面/电子记录，确保可追溯
3. **动态适配** - 实时跟踪政策变动，及时调整方案
4. **全面覆盖** - 管控贯穿"招聘→入职→在职→离职"全周期

# 模块2：政策层 - 全球用工合规政策

## 2.1 四大政策领域

| 领域 | 核心内容 |
|------|----------|
| **用工政策** | 劳动合同、试用期、工时、假期、解雇规则、反歧视法 |
| **薪税政策** | 最低工资、社保费率、个税计算、税务申报 |
| **移民政策** | 工作许可、签证类型、居留许可、签证续签 |
| **数据合规** | 员工信息收集、存储、使用、传输（GDPR等）|

## 2.2 合规管控流程

```
招聘阶段 → 入职阶段 → 在职阶段 → 离职阶段
   ↓          ↓          ↓          ↓
  JD合规    合同签署   薪税申报   离职结算
 渠道合规    入职手续   社保缴纳   证明开具
 背景调查    签证办理   绩效评估   竞业管理
```

## 2.3 政策变动应对

| 动作 | 说明 |
|------|------|
| 预警机制 | 对接属地律所/税务/移民局，月度更新清单 |
| 解读团队 | 明确政策变动对HR工作的影响 |
| 流程调整 | 及时调整HR流程、模板、方案 |
| 合规自查 | 政策变动后开展专项排查 |

# 模块3：组织层 - 海外组织架构与用工模式

## 3.1 用工模式对比

| 模式 | 定义 | 适用场景 | 优势 | 风险 |
|------|------|----------|------|------|
| **EOR名义雇主** | 第三方签署合同，承担合规责任 | 试水期、≤10人 | 快速落地、成本低 | 权责划分 |
| **IC独立承包商** | 签署服务协议，不直接管理 | 项目制、专业服务 | 用工灵活 | 事实雇佣认定 |
| **自雇佣** | 海外主体直接签署劳动合同 | 增长期/全球化 | 自主管理 | 合规复杂 |
| **外派员工** | 劳动关系在国内，海外履职 | 核心岗位、≤3年 | 保障核心业务 | 签证/税务 |

## 3.2 用工模式选择矩阵

| 因素 | EOR | IC | 自雇佣 | 外派 |
|------|-----|-----|--------|------|
| 有无海外主体 | 无 | 均可 | 有 | 均可 |
| 人员规模 | ≤10 | 任意 | ≥10 | 任意 |
| 成本 | 低 | 低 | 高 | 中 |
| 合规难度 | 低 | 中 | 高 | 中 |

## 3.3 权责划分要点

- **EOR**：服务商负责合规、薪税、签证；企业负责业务管理
- **IC**：明确交付成果、结算方式；避免日常考勤管理
- **自雇佣**：企业承担全部用工责任
- **外派**：国内承担劳动关系，海外负责日常管理

# 模块5：薪酬层 - 薪酬绩效与成本核算

## 5.1 薪酬方案设计

| 方案 | 适用场景 | 设计逻辑 |
|------|----------|----------|
| **派出国平衡法** | 短期外派（≤3年） | 派出国薪酬+外派激励+生活津贴 |
| **Local+法** | 长期全球化 | 东道国薪酬+专属福利津贴 |

## 5.2 核心成本构成

| 成本项 | 说明 |
|--------|------|
| 基本工资 | 月度/年度固定薪酬 |
| 绩效奖金 | 与业绩挂钩 |
| 法定福利 | 社保、养老金等 |
| 额外福利 | 住房、子女教育、商业保险 |
| 隐性成本 | 签证费用、搬迁费、税收成本 |

## 5.3 薪酬测算公式

```
总包成本 = 基本工资 × 12 + 绩效奖金 + 法定福利 + 额外福利 + 隐性成本

时薪成本 = 总包成本 ÷ (标准工时 × 52周)
```

# 模块6：区域层 - 核心国家合规要点

> 详细的国家政策文件见：`docs/知识库/区域政策/`

## 6.1 亚太区域

### 日本
- 社保合计：约41.3%（健康保险9.9%+护理保险1.62%+厚生年金18.3%+雇用保险1.35%+儿童育儿支援金0.23%）
- 最低工资：约1004日元/小时
- 年假：10-20天/年

### 澳洲
- 养老金：12%（雇主强制）
- 最低工资：22.63 AUD/小时
- 年假：20天/年
- 标准工时：38小时/周

## 6.2 欧洲区域

### 德国
- 最低工资：13.90欧元/小时（2026）
- 社保：约40.2%（雇主20%）
- 年假：24天/年
- 解雇通知期：4周-7个月

### 英国
- 最低工资：11.44英镑/小时
- 年假：28天/年

## 6.3 美洲区域

### 美国
- 最低工资：联邦7.25美元/小时（各州不同）
- 社保：6.2%+1.45%

- 最低工资：278.80墨西哥比索/天

- 标准工时：48小时/周

## 6.4 中东区域

### 阿联酋

- 无个人所得税
- 年假：30天/年

### 沙特

- 年假：21-30天/年

# 模块9：中小微出海方法论

> 详细方法论见：`docs/知识库/出海HR专业知识/成长型企业出海方法论与AI工具.md`

## 9.1 核心逻辑

> **"低成本、快迭代、强合规、小闭环"**

## 9.2 六大方法论

1. **战略前置** - 先定正确出海姿势
   - 三道自检关：产品关、现金流关、组织关
   - 四象限市场选型

2. **产品力打造** - 本土化微创新
   - 3个核心改动：功能适配、合规前置、消费习惯
   - SKU策略：1-3个核心SKU

3. **渠道获客** - 轻量化闭环
   - 渠道分阶段：0-1轻资产 → 1-10放大

4. **合规风控** - 生死红线
   - 知识产权、税务关税、产品认证、数据用工

5. **组织供应链** - 轻量化
   - 核心3人团队：项目负责人+运营+供应链
   - 非核心全外包

6. **长期增长** - 复利壁垒
   - 私域资产、细分品牌

## 9.3 八句心诀

1. 不贪大，先做细分，小池塘里做大鱼
2. 不冒进，先跑闭环，盈利之后再放大
3. 不内卷，价值定价，守住利润不低价
4. 不抄袭，微创新品，合规前置避大坑
5. 不重资，轻装前行，小步快跑快迭代
6. 不撒网，精准获客，私域沉淀复利来
7. 不蛮干，专业外包，核心环节自己抓
8. 不短视，长期主义，从小爆款到小品牌
