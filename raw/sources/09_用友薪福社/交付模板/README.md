# 交付模板

> 用友薪福社项目交付标准化模板集合

---

## 目录内容

| 文件 | 说明 |
|------|------|
| `交付交接流程.md/pdf` | 交付与交接标准流程 |
| `项目交付流程.md/pdf` | 项目交付各阶段SOP |
| `顾问协议模板.md/pdf` | 顾问合作协议（中/英文版） |
| `核心人才寻访服务合同.docx` | 猎头服务合同模板 |
| `承包商入职表单.xlsx` | 承包商入职信息收集 |
| `俄罗斯入职表单.xlsx` | 俄罗斯本地化合规入职表单 |
| `银行信息表单.xlsx` | 银行账户信息收集表 |
| `项目团队信息表.xlsx` | 项目团队成员信息 |
| `招聘需求收集表.xlsx` | 招聘需求标准化收集 |
| `项目交接清单.xlsx` | 交付物交接核对清单 |
| `发票模板.md/pdf` | 发票开具规范与模板 |

---

## 使用说明

1. **合同协议**：使用前请确认法律版本，适用于中国境内及目标国法律
2. **表单模板**：电子版（.xlsx）可直接分发给客户填写，纸质版（.pdf）可打印使用
3. **交接流程**：项目启动时与客户对齐交付里程碑，按节点提交交付物
