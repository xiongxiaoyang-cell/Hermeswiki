# 奇瑞出海HR合规项目探讨

> 来源：豆包会议纪要 | 2026-03-15

---

## 一、奇瑞国际「全球人力资源合规」详细需求

### 1. 企业基础背景

| 项目 | 详情 |
|------|------|
| 行业地位 | 国内车企出海第一梯队 |
| 年销量 | 300万辆+ |
| 海外销量占比 | 超50% |
| 海外机构 | 20+分支机构 |
| 海外用工规模 | 约1500人 |
| 现有资源 | 已与律所/合规机构合作，有各国劳动合规白皮书，但无法支撑业务实操 |

### 2. 表层直接需求（合规实操层面）

**欧盟等高合规区域精细化用工指引**

白皮书仅提供：固定期限/无固定期限合同的试用期时长

**奇瑞实际需要：**
- 试用期内：考核次数、考核形式、考核标准
- 考核结果：告知员工的合规流程
- 考核不通过：改进机制、整改流程
- 多次不通过后：试用期解除劳动合同的合规操作、解聘流程、风险防控

**全场景用工实操细则**
- 入职、离职、绩效、薪酬、解聘、补偿等日常管理动作
- 需要可直接执行的操作手册，而非法规条文

**陪跑式落地支持**
- 不满足于静态文档，需要专业方伴随业务、手把手指导落地

### 3. 中层管理需求（总部管控层面）

**解决中外管理冲突**
- 海外机构均配置：中方外派HR + 属地HR
- 双方在绩效、人员去留、用工管理上思路对立、操作冲突
- 集团总部无法判断属地HR提出的方案是否真实合规、是否合理

**外部权威智库参谋**
- 属地HR给出的方案往往对人效不友好、拉高成本
- 总部需要第三方专业机构验证属地HR方案的真实性与合理性，给出中立、专业、可落地的决策依据

**统一全球HR合规口径**
- 改变目前"各国单点、无体系"的状态，形成集团统一的海外HR合规管理标准

### 4. 深层底层需求（战略与经营层面）

**缺失全球HR合规顶层框架**
- 不清楚全球化HR合规应从哪些维度、哪些模块搭建，没有实施路径与优先级

**合规背后的经营痛点**
- 欧盟等区域法规限制严格（工时、加班、休假），人效远低于国内，成本居高不下
- 中外岗位、薪酬、编制无区分度，跨文化融合困难、内部矛盾大
- 单纯合规无法解决人效、成本、文化冲突，需要一体化解决方案

## 二、夏楠给出的逻辑框架、方法论

### （一）核心逻辑框架：双矩阵模型

#### 1. 第一矩阵：我方能力 × 客户需求深度

|  | 只要方向 | 需要方案 | 需要细化实操 | 需要带教 |
|--|---------|---------|------------|---------|
| **白皮书级** | | | | |
| **专业系统方案级** | | | | |
| **业务场景+落地级** | | | | |
| **全程陪跑级** | | | | |

**矩阵目的：**
- 精准界定双方合作边界、服务深度
- 明确项目报价依据、成本、交付标准
- 找到我方能力与客户需求的最佳契合点

#### 2. 第二矩阵：客户内部需求分级

- **维度1**：区域业务成熟度（欧盟核心区 → 其他成熟市场 → 新兴市场）
- **维度2**：需求刚性/紧急度（红线合规风险 → 日常管理痛点 → 优化提升）

### （二）整体解决方案方法论

#### 1. 底层认知
> **合规 ≠ 法条，合规 = 系统管理+实操+文化**

核心：合规是表象，背后是人力资源体系、人效、成本、跨文化的综合问题

#### 2. 逆向复用方法论
**外企入华逻辑 → 反向用于中企出海**

| 当年外企进入中国 | 现在中企出海 |
|----------------|-------------|
| 派核心高管+关键岗位 | 中方输出：管理者、关键技术/管理岗 |
| 基层本地化 | 属地招聘：基础操作岗 |
| 逐步文化融合 | 拉开岗位层级与薪酬区分度，减少文化冲突与对标矛盾 |

#### 3. 痛点穿透方法论

| 层次 | 问题 |
|------|------|
| 表面 | 试用期、解聘、绩效等合规操作 |
| 中层 | 中外HR对立、总部失控 |
| 底层 | 人效低、成本高、文化冲突、岗位/薪酬体系不匹配 |

#### 4. 落地实施方法论

- 不一次性覆盖所有国家
- 选定高合规、高痛点、高价值区域（如欧盟德/法/意/比）做样板国家
- 输出可复用的场景化实操模板，再复制到全球

#### 5. 智库支撑方法论

- 白皮书只有法条，没有实操套路、风险猫腻、本地潜规则
- 整合有海外实操经验的资深HR（尤其是外企出身），提供一线落地经验

---

### （三）引导/挖掘客户需求的专业方式

#### 1. 需求升维引导
从客户"要细化合规条文"，引导上升到：搭建全球HR合规顶层框架 → 明确模块、优先级、实施路径

#### 2. 痛点唤醒引导（用案例戳痛）
用外企退出中国N+6/N+7补偿案例，点醒客户：
- 属地HR会站在本地立场"放大合规要求"
- 中企将面临巨额成本、人效失控风险

#### 3. 价值锚定引导
- 客户只为落地、解决问题付费
- 白皮书/法条不值钱，场景化实操+陪跑+决策支持才是核心价值

#### 4. 路径引导：从模糊需求到可执行动作
- 客户："我们想做合规，但不知道怎么做"
- 引导：先选1个重点国家（如德国）→ 做1个核心场景（如试用期/解聘）→ 输出样板 → 客户认可后再扩体系

#### 5. 信任建立引导：甲方高管视角+实战案例
以多年甲方HRVP视角，用真实服务能源电池企业欧洲出海项目案例，建立专业权威

---

### （四）夏楠对本项目的核心建议总结

1. **先搭双矩阵**，定边界、定深度、定报价
2. **放弃粗颗粒白皮书**，做业务场景化实操合规
3. **以欧盟为突破口**，做样板示范
4. **合规+人效+薪酬+跨文化一体化解决**
5. **整合实战型海外HR智库**，支撑落地
6. **最终交付**：全球HR合规体系框架 + 重点国家实操手册 + 总部决策参谋服务
