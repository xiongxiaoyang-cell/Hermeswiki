## 二、各服务交付流程

### 1. EOR服务交付流程

```
1. 需求确认 → 2. 合同签署 → 3. 员工入职 → 4. 在职管理 → 5. 离职处理
```

**关键节点**：
- 合同周期：通常1年起
- 入职材料：5-10个工作日
- 每月薪酬：固定日期前完成

**服务优势**：
- 无需设立本地主体
- 快速合规用工
- 灵活调整规模

---

### 2. 海猎服务交付流程

```
1. JD沟通 → 2. 人才Mapping → 3. 候选人推荐 → 4. 面试安排 → 5. Offer跟进 → 6. 入职跟进
```

**关键节点**：
- Mapping报告：3-5个工作日
- 首批候选人：7-10个工作日
- 整体周期：1-3个月

**服务优势**：
- 全球人才网络
- 行业专业积累
- 跨文化招聘能力

---

### 3. HRO服务交付流程

```
1. 需求调研 → 2. 方案设计 → 3. 合同签署 → 4. 账户设立 → 5. 日常运营
```

**关键节点**：
- 社保账户设立：15-20个工作日
- 首月发薪：入职后次月
- 日常响应：24小时内

**服务优势**：
- 全流程合规
- 多国统一管理
- 专业政策解读

---

### 4. Payroll服务交付流程

```
1. 客户数据收集 → 2. 薪酬计算 → 3. 税务申报 → 4. 发薪处理 → 5. 报表交付
```

**关键节点**：
- 数据截止日：每月固定日期
- 发薪日期：可定制
- 报表交付：发薪后5个工作日内

**服务优势**：
- 多国统一发薪
- 税务合规
- 实时查询

---

### 5. 工签服务交付流程

```
1. 签证评估 → 2. 材料准备 → 3. 申请递交 → 4. 进度跟进 → 5. 签证实效管理
```

**关键节点**：
- 评估：1-2个工作日
- 办理周期：各国不同（2周-3个月）
- 续签提醒：到期前1个月

**服务优势**：
- 各国政策熟悉
- 成功率高
- 全程跟进
