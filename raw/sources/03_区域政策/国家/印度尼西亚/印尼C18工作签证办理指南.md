# 印尼C18工作签证办理指南 | 2026-03-28

**客户**：易宝支付（YeePay）
**对接人**：Luca Maggio
**EOR供应商**：Multiplier Technologies UK LTD

---

## 一、时间节点

| 节点 | 说明 |
|------|------|
| 员工入境日 | 请立即通知用友，开始办理流程 |
| 商务签最长停留 | 90天（从入境日起算） |
| 转签递交截止 | **签证生效第60天**（超过即转签失败） |
| 建议启动时间 | **入境15天内**（斋月期间办理效率拉长） |
| C18持有期间 | **不得离境**（离境即转签失败） |

> ⚠️ 斋月期间（3月）办理效率和时间周期都会拉长，必须尽早启动

---

## 二、关键规则

1. **入境即通知**：员工入境后立即通知用友，启动文件准备
2. **60天递交红线**：必须在签证生效第60天前完成转签递交
3. **15天保险启动**：斋月期间建议入境15天内启动转签流程
4. **不得离境**：从C18持有到拍照转签出Kitas期间，**员工不得离境**，离境即转签失败
5. **Kitas取得后自由**：取得Kitas（暂住证）后员工可自由离境

---

## 三、材料责任分工（ WanYan方 + 用友方）

### WanYan方（易宝支付）负责提供
1. 学士学位证书**扫描件**
2. 2寸电子版证件照2张（免冠、正面、面部清晰，双耳外露、不戴眼镜、白色背景，6个月内拍摄）
3. 护照**完整扫描件**（有效期>18个月，所有页面）
4. 附件Onboarding Form填写（用于劳动合同起草）

> 在提供上述扫描件后，用友同步进行材料公证翻译工作

### WanYan方协助提供（办理无犯罪记录）
- 带有印尼警方/相关主管部门印章和签名的临时居住证件
- 完整护照扫描件（所有页面）
- 完整签证页扫描件（两页）
- 父母全名及出生日期
- 配偶全名及出生日期（如有）
- 中国户籍地址
- 越南工作经历（如有）

### 用友方负责准备
1. 学士学位的**领事馆公证**
2. 工作经验的**领事馆认证**
3. 公司高管职位的**任命文件**
4. 安排员工完成**无犯罪记录**和**健康检查**

## 四、办理流程（建议）

```
入境印尼（第0天）
  ↓
立即通知用友，启动文件准备
  ↓
第15天内提交员工材料
  ↓
用友准备领事馆公证+认证+任命文件
  ↓
第60天前递交C18转签申请
  ↓
等待Kitas获批
  ↓
Kitas取得后可自由离境
```
