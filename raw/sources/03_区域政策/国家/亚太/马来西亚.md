## 最低工资政策

### 【2026年新增】2025年最低工资标准调整

**政策要点：**
- **生效时间：** 2025年1月1日
- **标准：** 全国统一最低工资提升至 **RM 1,700/月**
- **此前标准：** 2024年马来半岛为 RM 1,500/月，东马沙巴/沙捞越为 RM 1,350-1,400/月（分区域）

**调整详情：**

| 地区 | 2024年标准 | 2025年新标准 | 涨幅 |
|------|-----------|------------|------|
| 马来半岛 | RM 1,500/月 | RM 1,700/月 | +13.3% |
| 沙巴/沙捞越 | RM 1,350/月 | RM 1,700/月 | +25.9% |

> **信源：** The Star Malaysia, 2025年1月报道，马来西亚人力资源部（Kementerian Sumber Manusia）官方公告  
> **链接：** https://www.thestar.com.my/news/nation/2025/01/malaysia-hikes-minimum-wage-to-rm1-700

**对企业的影响：**
- 人力成本显著上升，尤其是东马地区（沙巴/沙捞越）涨幅更大
- 以100名基层员工（月薪RM 1,500）为例：月均人力成本增加约 RM 20,000，年增约 RM 240,000（不含其他配套社保费用）
- 建议企业重新审视薪酬架构，评估自动化替代可行性

**下一步调整动向：**
- 政府和工会持续呼吁2026年进一步上调最低工资至 RM 1,800-2,000
- 预计2026年中期将启动新一轮最低工资审查程序

**信源：** Malaysia Ministry of Human Resources (MOHR), 马来西亚政府官网  
**链接：** https://www.mohr.gov.my/

## 社保公积金(EPF)政策

### EPF 缴费比例（截至2026年3月）

马来西亚雇员公积金（Employees' Provident Fund, KWSP）是强制性的退休储蓄计划。

**标准缴费比例（2025-2026年无重大变化）：**

| 参保人类别 | 雇员缴纳 | 雇主缴纳 | 合计 |
|-----------|---------|---------|------|
| 马来西亚籍员工 | 11%（月薪上限 RM 5,000） | 12-13%（视行业） | 23-24% |
| 外籍员工 | 11%（月薪上限 RM 5,000） | 12-13%（视行业） | 23-24% |

**2025-2026年重要更新：**

> **【重要】EPF股息继续免税**（已确认延续至YA 2026）  
> 马来西亚政府确认，EPF账户持有人所获股息继续享有个人所得税免税待遇。  
> **信源：** The Star Malaysia, 2026年3月报道  
> **链接：** https://www.thestar.com.my/news/nation/2026/03/epf-dividends-tax-exempt

**EPF 账户利率参考（近年数据）：**
- 2023年：5.0%-5.35%（传统账户 vs 伊斯兰账户）
- 2024年：约4.5%-4.75%
- 2025年：约4.0%-4.5%（具体以KWSP官方公布为准）

**中资企业注意事项：**
1. 雇主缴纳比例因行业而异，制造业通常为13%，服务业约12%
2. 超出缴费上限（RM 5,000/月）的薪资部分不计入EPF缴费基数
3. 外籍员工与本地员工适用相同EPF缴费比例
4. 雇主需每月按时向KWSP缴款，逾期将产生罚款

**信源：** 马来西亚雇员公积金局（KWSP）官网  
**链接：** https://www.kwsp.gov.my/

## 就业准证(EP)政策

### 【2026年新增】Employment Pass (EP) 最新要求

马来西亚就业准证（Employment Pass）是发给外籍专业人士的工作许可，由马来西亚移民局（JIM）负责审批。

**最新薪资门槛（2025年起生效）：**

| EP类别 | 最低月薪要求 | 备注 |
|--------|------------|------|
| EP I（专业人才） | **RM 10,000/月** | 主要针对高级管理层、专业人士 |
| EP II（技术工人） | **RM 5,000/月** | 技术岗位 |
| EP III（临时工） | **RM 3,000/月** | 短期合同 |

> **信源：** Malaysia Digital Economy Corporation (MDEC) 及多家媒体报道，2025年EP薪资门槛调整  
> **链接：** https://www.malaymail.com/news/malaysia/2025/02/malaysia-employment-pass-new-salary-threshold

**关键政策变化：**

1. **薪资门槛大幅提高**：EP I 的最低月薪要求从 RM 5,000-7,000 大幅提升至 RM 10,000，主要目的是吸引更高素质的专业人才
2. **劳动力市场测试（LMTT）**：雇主须证明已在本地媒体刊登招聘广告，无法聘到合适本地员工
3. **配额限制**：某些行业对外国人就业比例有严格限制
4. **EP有效期**：通常为2-5年，到期可续

**申请流程与时长：**
- 标准审批时间：约4-8周（不含文件准备）
- 快速通道（经MDEC或工业特区）：约2-4周
- 必须通过 MyExperience 系统进行申请

**中资企业合规要点：**
1. 外籍员工入职前必须先取得EP，非法就业将被严厉处罚（罚款、监禁、驱逐出境）
2. 雇主需为每位外籍员工向移民局登记
3. EP持有者的配偶及子女可申请家属准证（Dependant Pass）
4. 部分行业（建筑、制造、农业、服务业）对外籍劳工有特殊配额，须通过人力资源发展局（HRD Corp）办理

**信源：** Malaysia Immigration Department 官网  
**链接：** https://www.imi.gov.my/

## 工伤保险(SOCSO/EIS)

### 马来西亚社会保障（SOCSO）与就业保险（EIS）

马来西亚建立了完善的社会保障体系，包括工伤保障和失业保障：

**SOCSO（Social Security Organization）- 工伤保险：**

| 项目 | 详情 |
|------|------|
| 全称 | Pertubuhan Keselamatan Sosial (PERKESO) |
| 缴费主体 | 雇主（全额，雇员不需缴纳） |
| 缴费比例 | 雇员月薪的 1.25%-1.75%（视行业风险等级） |
| 最高月薪基数 | RM 5,000 |
| 保障范围 | 工伤、职业病、残疾赔偿、死亡赔偿、康复费用 |

**EIS（Employment Insurance System）- 就业保险：**

| 项目 | 详情 |
|------|------|
| 全称 | Sistem Insurans Buruh (SIB) / EIS |
| 生效时间 | 2018年1月1日 |
| 缴费主体 | 雇主（RM 0.2%/月薪）+ 雇员（RM 0.1%/月薪） |
| 月薪上限 | RM 5,000 |
| 保障期限 | 最长6个月（按失业前月薪比例发放） |
| 适用条件 | 非自愿失业（被裁退、公司倒闭） |

**2025-2026年政策稳定性：**
- SOCSO/EIS 缴费比例和计算方式在2025-2026年无重大调整
- 雇主需按月通过 SOCSO 官方系统 e-Caruman 缴款

**中资企业合规要点：**
1. 所有在马来西亚就业的雇员（含外籍）均须强制登记 SOCSO
2. 未登记参加 SOCSO 属于违法，可被罚款或起诉
3. 工伤申报须在规定时间内完成（事故发生后 10 个工作日内）
4. EIS 由雇主和雇员共同缴费，雇主需代扣代缴

**信源：** 马来西亚社会保障机构（PERKESO）官网  
**链接：** https://www.perkeso.gov.my/

## 税制变化

### 【2026年新增】马来西亚2025-2026年重要税制更新

#### 一、股息税（Dividend Tax）

**【2025年预算案引入，2026年重要更新】**

| 项目 | 详情 |
|------|------|
| 股息免税额 | 个人股东股息所得中，首 RM 100,000 可获 50% 免税（2025-2026年维持） |
| 股息附加税 | 超出 RM 100,000 的股息，按 **50% 税率** 征收（即有效税率约50% x 适用税率） |
| EPF股息 | EPF 账户持有人所获股息继续**全免**个人所得税 |
| 生效时间 | 适用于2025年度评估（YA 2025，即2024年所得） |

> **信源：** The Star Malaysia, 2026年3月确认EPF股息免税政策延续  
> **链接：** https://www.thestar.com.my/news/nation/2026/03/epf-dividends-tax-exempt

#### 二、LLP利润分配税（Limited Liability Partnership Distribution Tax）

**【2026年新增】**
- 自YA 2026（即2026年度评估）起，LLP（有限责任合伙企业）的利润分配将被征收 **2% 的最终税**
- 此前LLP利润分配享受免税待遇
- 这对通过LLP架构进行利润分配的投资者有重大影响

#### 三、个人所得税（Individual Income Tax）

**2025年马来西亚个人所得税税率结构（参考）：**

| 应税收入区间（RM） | 税率 |
|-------------------|------|
| 0 - 5,000 | 0% |
| 5,001 - 20,000 | 1% |
| 20,001 - 35,000 | 3% |
| 35,001 - 50,000 | 6% |
| 50,001 - 70,000 | 11% |
| 70,001 - 100,000 | 19% |
| 100,001 - 400,000 | 25% |
| 400,001 - 600,000 | 26% |
| 600,001 - 1,000,000 | 28% |
| 1,000,001 - 2,000,000 | 29% |
| 2,000,001 以上 | 30% |

**就业收入免除额（Employment Income Exemption）：**
- 2025年起，退休金、回教贡献金（Zakat）扣除规定有调整
- 详细请咨询专业税务顾问

#### 四、其他重要税制

| 税种 | 说明 |
|------|------|
| 企业所得税 | 标准税率 24%（中小型企业有优惠） |
| 销售税（SST） | 6%-10%（大部分商品和服务） |
| 服务税 | 6%-8% |
| 预扣税 | 非居民股息 0%、利息 15%、特许权使用费 10%（有税收协定优惠） |
| 数字服务税 | 6%（针对数字服务平台） |

#### 五、中资企业税务合规要点

1. **转让定价**：马来西亚要求关联交易按公平交易原则定价，大额跨境交易需准备转让定价文档
2. **常设机构（PE）风险**：员工在马长期工作可能被认定为在马构成PE
3. **预扣税申报**：向非居民支付款项时需按规定代扣代缴预扣税
4. **税务亏损结转**：企业亏损可在未来最多7年内结转抵扣

**信源：** 马来西亚内陆税务局（Lembaga Hasil Dalam Negeri, LHDN）官网  
**链接：** https://www.hasil.gov.my/

## 外籍劳工政策

### 【2026年新增】马来西亚外籍劳工最新政策

马来西亚严重依赖外籍劳工，尤其是在建筑、制造、种植业和服务业。2025-2026年政策有以下重要更新：

#### 一、建筑行业外劳政策

| 项目 | 政策内容 |
|------|---------|
| 外劳上限 | 建筑行业外籍劳工比例可达 1:4（即1名本地工人可配4名外劳） |
| 配额申请 | 通过 PUSPASTEMAN 系统在线申请 |
| 允许来源国 | 孟加拉国、印度尼西亚、尼泊尔、巴基斯坦、越南等 |
| 最低工资要求 | 建筑工人最低月薪 RM 1,700（适用最新最低工资标准） |

> **信源：** Malaysia Ministry of Human Resources (MOHR) 关于建筑业外劳配额公告

#### 二、制造业外劳政策

| 项目 | 政策内容 |
|------|---------|
| 外劳上限 | 制造业外籍劳工比例通常为企业总劳动力的 **30%-40%**（视子行业） |
| 高科技行业 | 高科技制造业可申请更高比例外劳配额 |
| 允许来源国 | 主要为孟加拉国、印尼、尼泊尔、缅甸、越南 |
| 特殊政策 | 2025年起，政府对特定子行业（如电子电气）的外劳配额进行了优化调整 |

#### 三、农业（种植业）外劳政策

| 项目 | 政策内容 |
|------|---------|
| 外劳上限 | 种植业严重依赖外劳，允许较高比例（部分可达 80-90%） |
| 来源国 | 主要为印尼（沙巴/沙捞越）、孟加拉国 |
| 政策 | 政府推出"合法化计划"（外劳漂白计划），允许无证外劳通过验证程序获得合法身份 |

#### 四、2025-2026年政策动向

1. **孟加拉国 MOU 签署**：马孟两国政府持续更新外劳引进凉解备忘录（MOU）
2. **外劳人头税**：雇主需缴纳外劳人头税，费用约 RM 1,850-10,000/年/人（视行业和来源国）
3. **强制体检**：所有外籍劳工须定期进行健康检查
4. **"漂白计划"延长**：政府多次延长非法外劳自愿遣返及合法化计划期限

#### 五、中资企业合规要点

1. **配额申请必须提前规划**：外劳配额申请耗时较长，建议提前6-12个月启动
2. **通过正规渠道引进**：必须通过马来西亚政府认可的职业介绍所（licensed recruitment agencies）
3. **人头税预算**：人头税是重大成本，需纳入整体人力成本预算
4. **外劳宿舍合规**：须遵守 Workers' Minimum Standards of Housing and Amenities Act 1990 (Act 446)
5. **不得雇佣非法外劳**：一经查实，雇主将面临严厉处罚（罚款、监禁、禁止雇佣外劳）

**信源：** 马来西亚移民局（IMI）、人力资源发展局（HRD Corp）  
**链接：** https://www.imi.gov.my/

## Gig Workers Act 2025

**【2026年新增重要立法】**

| 项目 | 内容 |
|------|------|
| 法案名称 | Gig Workers Act 2025 |
| 预计生效时间 | 2026年3月 |
| 适用范围 | 平台经济工人（网约车、外卖、自由职业平台等） |
| 主要内容 | 灵活工作者权益保护、社保纳入、最低工资保障 |

> **信源：** 马来西亚人力资源部（MOHR）公告

- 可能增加平台企业人力成本
- 中资企业如使用平台用工，需关注合规要求

## 劳动法更新

### 2025-2026年劳动法立法动态

马来西亚主要的劳动法律框架包括：

| 法律法规 | 说明 |
|---------|------|
| Employment Act 1955（雇用法令） | 规范所有雇主与雇员关系的基础法律 |
| Industrial Relations Act 1967 | 规范劳动关系和工会 |
| Trade Unions Act 1959 | 规范工会组织 |
| Occupational Safety and Health Act 1994 (Act 514) | 职业安全与健康 |
| Workers' Minimum Standards of Housing and Amenities Act 1990 (Act 446) | 外籍劳工宿舍标准 |

**2025-2026年重要立法动态：**

1. **Employment Act 修订讨论持续**：
   - 政府持续研讨Employment Act的修订，以更好地保护平台经济工人（gig workers）和弹性工作安排
   - 截至2026年3月，尚无重大修订法案正式通过，但业界预期修订将分阶段推进

2. **弹性工作安排（Flexible Working Arrangements）**：
   - 2024年正式通过 Employment (Amendment) Act 2023，要求雇主书面记录弹性工作申请并予以考虑
   - 2025-2026年：该政策执行力度加强，雇员申请弹性工作安排的权利得到更充分保障

3. **同工同酬（Equal Pay）**：
   - 政策层面推动同工同酬，但尚无专门的同工同酬立法
   - 禁止基于性别歧视报酬差异的规定已纳入 Employment Act

4. **产假与陪产假**：
   - 产假：98天（已与国际劳工组织公约C183一致）
   - 陪产假：2024年起增加到 7 天（此前为 3 天）

5. **外劳管理强化**：
   - 外劳宿舍合规检查频率增加，违规罚款提高
   - 建筑和制造业外劳宿舍必须符合 Act 446 标准
