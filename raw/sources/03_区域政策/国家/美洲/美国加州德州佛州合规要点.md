## 一、offer与预入职合规要求

### 1.1 offer格式要求
| 项目 | 要求 |
|------|------|
| 工资标准 | 必须明确 |
| 支付周期 | 必须明确 |
| 加班规则 | 必须明确 |
| 带薪休假条款 | 必须明确 |
| 保存年限 | **4年** |

### 1.2 竞业协议（Covenants not to compete）

| 州 | 规则 | 法律依据 |
|----|------|----------|
| **加州** | 非竞争协议**完全无效** | Business & Professions Code §16000 |
| **加州** | 2026年1月起禁止"留任或赔付"协议（AB 692），不得要求员工返还培训费、搬迁费等费用 | AB 692 |
| **德州** | 允许但需满足"合理"三要素：时间（通常≤1年）、地域范围（与业务相关）、保护合法商业利益 | — |
| **佛州** | 仅对高薪员工（>2倍当地平均工资）可执行 | — |

### 1.3 零售员工加班豁免复核（7(i)）
导游人员可申请加班豁免，但需通过：
- 零售/服务机构资格测试
- 薪资率测试
- 佣金占比测试

> **要求**：每季度复核，以保证豁免有效性

## 二、加班规则（加州 vs 德州/佛州）

### 2.1 加州：双重标准
| 条件 | 倍数 |
|------|------|
| 日工作超8小时 或 周工作超40小时 | 1.5倍 |
| 第6天首8小时 | 1.5倍 |
| 第6天超12小时 | 2倍 |
| 第7天（无论时长） | 2倍 |

> ⚠️ **加州特有要求**：加班规则必须在offer中**明示**

### 2.2 德州 / 佛州：单一标准
- 仅按**周工作超40小时**计算加班
- 倍率：1.5倍工资
- **无日加班要求**

## 三、管理人员职责测试（Exempt vs Non-exempt）

### 3.1 测试要件
管理人员豁免资格需满足以下**全部条件**：

| 要件 | 说明 |
|------|------|
| 管理范围 | 管理企业或常设部门 |
| 人员指导 | 直接指导 ≥2名全职员工 |
| 人事决策 | 具有人事决策权 |
| 独立判断 | 具有独立判断权 |

### 3.2 证明文件要求
| 项目 | 要求 |
|------|------|
| 保存年限 | **4年** |
| 记录内容 | 每日工作内容、时间分配、任务性质 |
| 核心证明 | 管理职责占比必须 **> 50%** |

## 四、法定通知签收制度

### 4.1 入职必须提供文件
- 《员工权利告知》
- 《带薪病假权利》
- 《反歧视通知》
- **2026年新要求**：紧急联系人授权记录

### 4.2 签收记录管理
| 项目 | 要求 |
|------|------|
| 保存年限 | **3年** |
| 档案绑定 | 签收记录与员工档案**永久绑定** |
| 缺失后果 | **无法完成入职流程** |

### 4.3 合规支持
系统需支持：
- 法定通知电子推送
- 电子签收
- 签收记录与员工档案自动绑定

## 五、州际差异总结

| 维度 | 加州 | 德州 | 佛州 |
|------|------|------|------|
| 竞业协议 | **完全无效** | 合理三要素 | 高薪员工可执行 |
| 加班标准 | **双重标准**（日+周） | 单一标准（周） | 单一标准（周） |
| 加班明示 | **必须** | 无 | 无 |
| 培训费返还 | **禁止**（AB 692） | 允许 | — |
