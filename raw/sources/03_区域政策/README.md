## 信息发布原则

⚠️ **本知识库政策信息均需交叉验证信源：**
1. 政策目标国官方（劳工部/社保局/税务局官网）
2. 中国官方（人社部、商务部、税务总局官网）
3. 国际组织（ILO、世界银行）
4. 权威行业媒体（第一资源、HRflag等）
