## 🔴重大更新判断标准（6条）

满足任意一条即🔴：
1. 法规废立（法律/政策新增或废除，含草案落地前夕）
2. 数值变化（最低工资/社保费率等，含"预计上调X%"）
3. 适用范围变化（外籍劳工政策范围任何扩大或缩小）
4. 限制变化（签证/工作许可要求任何收紧或放宽）
5. 外籍政策关联（所有涉及外籍劳工的签证/社保/工作许可变化）
6. 生效时间≤30天

**假阳性防范**：必须有实际数值/日期/条款；标题含2026/Guide/Report不足以触发🔴

### ⚠️ Known Issue: Authorization Header Bug

**问题**：`feishu_send_file()` 和 `feishu_send_text()` 中 Authorization header 写作 `Authorization: Bearer ***`（字面量 `***`），而正确写法应为 `Authorization: Bearer {token}`（token 变量插值）。这会导致发到龙虾群的 PDF 和文字消息返回 401 认证失败，但发到政策日报群的 PNG 图片（走 `feishu_send_image()`）不受影响。

**症状**：日报 PNG 正常发到政策日报群，但龙虾群收不到文字/PDF。

**修复方法**：
```python
# 错误（字面量 ***）：
-H",f"Authorization: Bearer ***         "

# 正确（变量插值）：
-H",f"Authorization: Bearer {token}"
```
搜索 `Bearer \*\*\*` 找到所有出现位置并修复。
