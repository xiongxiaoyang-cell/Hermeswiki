## 一、产品定位与名称

### 1.1 产品定名

**「出海领航」年度战略陪跑计划**

英文名：**Go-Global Annual Strategic Partnership**

定位语：**"让每一分出海投入，都有人陪着你走"**

### 1.2 顶层定位

| 维度 | 说明 |
|------|------|
| 产品类型 | 高端年度制咨询陪跑服务（非课程、非训练营） |
| 核心逻辑 | **咨询为主，工具为辅** —— 不是教你学，是帮你做 |
| 适合客户 | 有出海战略意图、但缺乏落地路径和执行团队的成长型企业 |
| 价值主张 | 用友全球资源 + 迅合专业团队 + 驻场式陪伴 = 出海真正落地 |

### 1.3 产品组合示意

```
698,000 年度陪跑咨询产品
├── 10次入企1V1深度咨询（每月约1次）
├── 4次季度战略复盘会（每次半天，团队参与）
├── 年度企业出海诊断报告（1份）
├── 出海资源对接服务包（含大使馆、用友全球网络）
├── 工具方案建议包
└── 专属客户成功经理（CSM）+ 专家顾问团
```

## 二、服务内容详细拆解

### 2.1 月度入企1V1咨询（全年10次）

**形式：** 每次1天（6-8小时），顾问团队入驻企业，深度调研+策略研讨

| 月份 | 主题 | 核心交付物 |
|------|------|-----------|
| 第1个月 | 出海战略启动诊断 | 《企业出海能力诊断报告》 |
| 第2个月 | 目标市场选择与可行性分析 | 《目标市场进入可行性报告》 |
| 第3个月 | 商业模式本地化设计 | 《商业模式本地化适配方案》 |
| 第4个月 | 合规架构搭建 | 《目标国合规架构建议书》 |
| 第5个月 | 财务与税务筹划 | 《跨境税务筹划方案》 |
| 第6个月 | 团队搭建与本地化 | 《本地团队搭建路径图》 |
| 第7个月 | 合作伙伴/渠道策略 | 《渠道与合作伙伴策略报告》 |
| 第8个月 | 品牌与市场进入 | 《市场进入执行方案》 |
| 第9个月 | 数字化工具选型 | 《工具建议与实施路线图》 |
| 第10个月 | 风险管控与应急预案 | 《风险管控手册》 |

> **注：** 第11-12个月合并至Q4战略复盘，不单独安排入企，以确保全年服务完整性。

**每次入企咨询标准流程（约7小时）：**

```
09:30-10:00  企业核心团队访谈（CEO/业务负责人/财务负责人）
10:00-11:30  业务现状深度调研（数据、流程、痛点）
11:30-12:30  专家内部策略研讨
13:30-15:30  策略方案现场共创
15:30-16:30  优先级排序与下阶段行动计划
16:30-17:00  书面记录与双方签字确认
```

---

### 2.2 季度战略复盘会（全年4次）

**形式：** 每次半天（3-4小时），线下/线上均可，建议企业核心团队3-5人参加

| 季度 | 时间节点 | 主题 |
|------|---------|------|
| Q1复盘 | 第3个月末 | 战略落地进度评估 + Q2策略调整 |
| Q2复盘 | 第6个月末 | 半年度复盘 + 下半年重点重新排序 |
| Q3复盘 | 第9个月末 | 前三季度冲刺评估 + 年度目标校验 |
| Q4复盘 | 第12个月末 | 年度总结 + 下年度战略规划启动 |

**每次季度复盘会标准议程：**

```
1. 企业自评报告汇报（企业方30分钟）
2. 关键指标数据分析（顾问方20分钟）
3. 策略执行偏差分析（30分钟）
4. Q+N策略调整方案共创（60分钟）
5. 下季度重点工作确认（20分钟）
6. 会议纪要 & 行动清单确认（10分钟）
```

```
五、12个月提升路径图
```

### 4.3 与迅合的分佣模式

**背景假设：** 迅合负责客户推荐/渠道，迅合负责部分课程内容，我们负责整体产品交付与顾问管理。

| 方案 | 描述 | 迅合收益 | 我们收益 | 备注 |
|------|------|---------|---------|------|
| 方案A（推荐） | 迅合负责销售，我们负责交付 | 销售佣金 20%（139,600） | 80%（558,400） | 迅合开票给客户 |
| 方案B | 联合服务，共同交付 | 利润分成 35%（107,800） | 65%（200,200） | 双方各自承担对应成本 |
| 方案C | 迅合只提供渠道线索，不参与交付 | 渠道佣金 15%（104,700） | 85%（593,300） | 轻量合作 |

**建议采用方案A**，理由：
- 迅合作为品牌背书方，销售佣金合理
- 20%佣金在渠道合作中属于正常区间（15-25%）
- 我们的毛利空间仍有44%，具备可持续性

**方案A利润情况（按方案A）：**

| 项目 | 金额 |
|------|------|
| 客户付款 | 698,000 |
| 迅合佣金（20%） | -139,600 |
| 我们收款 | 558,400 |
| 我们成本 | -390,000 |
| **我们净利润** | **168,400** |
| **我们净利率** | **约30.2%（按到账558,400计）** |

## 五、与69800产品的升级路径

### 5.1 产品关系定位

```
69800（私董会年度会员）
    │
    ├── 已有出海认知基础
    ├── 有圈子、有圈子信任
    └── 适合：想了解出海，但还没下定决心
           ─────────────────────────────
           ⬆ 升级机会点
           ─────────────────────────────
           │ "你已经在私董会了解了出海方向，
           │  现在需要的是真正落地陪跑"
           ▼
698000（年度战略陪跑）
    │
    ├── 10次入企 + 季度复盘 = 持续深度陪伴
    ├── 专属团队 + 专属资源
    └── 适合：已确定出海战略，需要执行落地支持
```

### 5.2 升级识别信号

以下私董会会员特征出现时，主动触发升级触达：

| 信号类型 | 具体表现 |
|---------|---------|
| 业务层面 | 开始谈论具体目标市场、开始招聘外贸负责人、拿到进出口资质 |
| 行为层面 | 多次参加出海相关分享、私下询问出海落地细节 |
| 时机层面 | 年底战略规划期、融资完成、有出海预算审批 |
| 危机层面 | 海外客户找上门但不知如何接、已有海外订单但合规有问题 |

### 5.3 升级话术

#### 场景一：私董会活动中的自然切入

> "王总，我注意到您最近几次活动都在问东南亚市场的事。您目前是已经有具体的出海想法了，还是还在探索阶段？
>
> 如果是已经在规划的话，我们最近新推了一个「出海领航」年度陪跑项目——不是课程，是顾问团队每个月入驻企业一次，全年10次深度咨询，外加季度复盘会，还有用友全球网络的资源对接。
>
> 您现在私董会的学习是帮您打开视野，而陪跑项目是帮您真正落地。我觉得对您现在的阶段，可能后者更实用。您有兴趣的话，我可以给您发个详细方案，我们约个时间专门聊一聊？"

#### 场景二：一对一私聊升级

> "李总，上次您说到您想探索东南亚市场，现在进展怎么样了？
>
> 我想直接问您一个问题：如果现在有一个服务，能让您在接下来一年里，每个月有专家团队到您企业来一天，陪着您把出海这件事一步一步做下来——从市场选择、合规架构、财务筹划，到本地团队搭建——您觉得这个对您有价值吗？
>
> 这个项目我们叫「出海领航年度战略陪跑」，定价698,000，包含10次入企咨询、4次季度复盘，还有资源对接服务。
>
> 我知道价格不低，但我跟您算笔账：您现在如果单独请一个出海负责人，年薪至少40-60万，还不一定专业。而这个项目是整个专家团队在支持您，从战略到执行全程陪跑。
>
> 您看这周我们约个时间，我带着方案来给您详细讲讲？"

#### 场景三：针对已有出海订单的企业

> "张总，听说您最近接到了第一笔海外订单，恭喜！这其实是最好的时机来认真规划一下出海节奏。
>
> 我见过很多企业，第一单靠运气接到了，但第二单、第三单跟不上——问题往往不在业务本身，而在背后的合规、财务、团队没有准备好。
>
> 我们新推出的「出海领航」年度陪跑项目，核心就是帮您把出海的基础设施打牢。您现在加入，到年底之前，您的合规架构、财务筹划、本地化团队这些关键节点都能落地完成。
>
> 我建议您认真考虑一下，这个时间点加入是最划算的。"

### 5.4 升级时间窗口

| 最佳触达时机 | 原因 |
|------------|------|
| 年底战略规划期（11-1月） | 企业制定次年预算，出海预算开口期 |
| 融资完成后 | 企业有资金，有扩张冲动 |
| 参加完出海主题私董会活动后 | 热度最高，决策意愿最强 |
| 重大业务事件后（拿到海外订单/遇到合规问题） | 需求被激活 |

## 八、风险提示与合规边界
