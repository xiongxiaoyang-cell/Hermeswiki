## 一、产品整体形态

### 1.1 产品命名

**主推名称：**「航海家私董会」—— 出海实战私董班  
**副标题：** 69800 / 人 | 5次私董会 × 2天/次 | 全年复训权

**品牌调性：**  
- 定位高端决策圈层（"这是给有航海野心的人开的船"）  
- 区别于市场通用"私董会"叫法，「航海家」更具身份认同感和故事性  
- 对外传播语：**"跟已经到过彼岸的人，照亮你的航海图"**

**备选名称（可注册商标用）：**  
- 「领航会」  
- 「彼岸私董会」  
- 「舵手私董会」  

---

### 1.2 目标人群画像

| 维度 | 描述 |
|------|------|
| **阶段** | 0→1：已有产品/服务，正在或准备启动出海，从0到1验证商业模式 |
| **身份** | 企业创始人 / 联合创始人 / 实际控制人 |
| **资金门槛** | 已投入或准备投入出海业务 ≥ 50万人民币 |
| **年营收** | 0 – 5000万（越早期越饥渴，辅导价值越大） |
| **地域** | 以华东、华南为主，辐射全国 |
| **痛点** | 不知道从哪开始；做了很多试错；踩坑成本极高；没有同行者 |
| **心理** | 孤独、急切、愿意为"过来人经验"付溢价 |

**典型画像：**  
> "王总，深圳某工贸一体企业老板，45岁，做亚马逊3年亏损200万，不知道问题出在哪，参加航海家私董会后找到流量闭环漏洞，第2期扭亏。"

## 三、定价方案

### 3.1 定价逻辑

**69800元/人**的定价依据：

| 维度 | 数据/逻辑 |
|------|-----------|
| **成本逻辑** | 5次 × 2天 = 10天线下，10-15人/期，合计100-150人天；锴哥/专家时间成本即超30万，加上场地/物料/服务，69800覆盖合理利润 |
| **对标逻辑** | 市场同类私董会（如青腾汇、混沌商学院）定价5-15万；「航海家」有出海垂直稀缺性，69800处于中偏高但可接受区间 |
| **价值锚定** | 学员平均踩坑成本远超69800（一次失败选品/商标/平台可能损失50万+），投资10%于认知提升是理性决策 |
| **心理定价** | 69800 < 10万，低于心理门槛；接近7万，有"不到7万"的感觉 |
| **升级锚定** | 69800是698000的1/10，自然形成"先体验再升级"的心理锚点 |

| **三人团报价** | 62800元/人 | 同团3人同时付款 |

> **建议：** 首发期用早鸟59800快速锁定种子用户（前15名），30天后恢复原价69800制造紧迫感。

## 四、运营流程

### 4.1 招生方式

| 渠道 | 具体动作 | 责任人 |
|------|---------|--------|
| **迅合私域转化** | 迅合现有学员社群、直播、短视频流量，引导报名 | 迅合运营 |
| **IP联名推广** | 锴哥公众号/视频号/朋友圈预告，限时报名通道 | 锴哥团队 |
| **定向邀约** | 根据迅合CRM筛选0→1阶段企业主，1对1私信邀约 | 双方联合 |
| **口碑裂变** | 老学员推荐减费激励机制 | 迅合CRM |
| **线下活动** | 沙龙/展会设展台，现场咨询报名 | 迅合市场 |

**招生节奏（首期为例）：**

```
T-30天：开放报名（早鸟价59800）
T-20天：早鸟价截止倒计时
T-15天：满10人即开班，发布开班通知
T-7天： 关闭报名，准备开班物料
T：     第1期开班
```

**招生文案核心钩子：**  
> "我已经花了500万踩过的坑，你不需要再踩一遍。5次私董会，10天，锴哥带你把出海从0到1走一遍。"

### 4.2 城市安排逻辑

| 期次 | 推荐城市 | 选择理由 |
|------|---------|---------|
| 第1期 | 上海 | 便于华东学员，迅合总部资源支撑 |
| 第2期 | 深圳 | 华南卖家核心地，亚马逊生态最密集 |
| 第3期 | 杭州 | 品牌出海/TikTok/跨境电商人才聚集 |
| 第4期 | 北京 | 融资/资本/政策资源最集中 |
| 第5期 | 待定 | 可选出海目的地（新加坡/迪拜/越南），做海外游学版 |

**原则：**  
- 每次尽量选择不同城市，增加新鲜感和本地资源渗透  
- 场地优先选择精品酒店会议室（控制成本，控制在3000-5000元/人/天含茶歇）  
- 场地需提前30天确认，确保私密性（不对外开放）

### 4.3 复训机制

**全年5次复训权：**  
- 学员购买69800后，当年内5次私董会均有权参加（不限期次，不限城市）  
- 复训学员座位在后排，不参与私董案主环节（但可选旁听私董诊断）  
- 复训提前7天预约，迅合提前14天发布当期安排  
- 复训学员可参加当期的社交活动，但不占用新学员名额

**复训利用策略：**  
- 建议学员带团队核心高管一起参训（副驾驶名额，享老学员价）
- 每期预留2-3席给复训学员，平衡新老学员体验

### 4.4 与迅合的配合方式

| 模块 | 迅合负责 | 锴哥/专家负责 |
|------|---------|-------------|
| 产品设计 | 提供数据支持、竞品参考 | 核心内容设计、导师安排 |
| 招生运营 | 招生渠道、CRM管理、收款 | 宣传素材、IP背书 |
| 会务执行 | 场地预订、物料制作、餐饮 | 核心课程讲授 |
| 课后跟进 | 学员群运营、复训预约 | 专家答疑、1对1咨询 |
| 持续迭代 | 学员满意度调研 | 课程内容更新 |

**权责边界：**  
- 迅合负责运营和后勤，锴哥/专家负责内容和教学质量  
- 定价策略双方共同确定，招生收由迅合统一管理  
- 锴哥对课程质量和核心内容有一票否决权

## 六、FAQ（常见问题）

### 6.1 产品相关

**Q：69800包含什么？**  
A：5次私董会（每次2天）、全年5次复训权、专属学员群、课件工具包、1次1对1专家咨询（1小时）、迅合公开课优先权。

**Q：锴哥会全程主讲吗？**  
A：5期中，锴哥主持第1期（开班）、第3期（品牌）和第5期（毕业），第2期和第4期由锴哥审核认证的专家团队讲授，内容质量由锴哥把关。

**Q：每期有多少人？**  
A：每期10-15人，精品小班制，确保私董案主环节的质量。

**Q：如果某一期来不了，可以补课吗？**  
A：可以。当年5次会期均可预约复训，只需提前7天告知我们会务组即可。

**Q：课程内容每年都一样吗？**  
A：课程框架固定（5期主题），但案例、案主问题、市场动态每年更新，确保与时俱进。

### 6.2 报名相关

**Q：报名条件是什么？**  
A：企业创始人/联合创始人，正在或准备出海业务。建议已有一定投入（时间或资金），纯小白暂不接受。

**Q：怎么报名？**  
A：联系迅合课程顾问 → 顾问审核资格 → 付款确认 → 发送入学通知书。

**Q：可以企业转账吗？**  
A：支持对公转账，发票由迅合或其合作机构开具（普票/专票均可）。

**Q：付款后可以退款吗？**  
A：开班前7天申请退款，全额退款；开班前3-7天，退款70%；开班后，不退款，但可转让名额。

### 6.3 升级相关
