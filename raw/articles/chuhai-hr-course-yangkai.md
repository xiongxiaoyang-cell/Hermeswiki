## 一、出海背景与驱动力

### 中企出海行业分布（2024年末）
- 中国出海企业约**3.4万家**，近八成集中在：制造（33.3%）、批发零售（20.9%）、租赁和商务服务、信息软件技术服务
- 在全球190个国家和地区设立企业达**5.2万家**，遍布全球80%以上国家地区

### 出海动因矩阵
| 国内痛点 | 海外机会 |
|---------|---------|
| 价格战打到地板价，利润微薄 | 价值定价，溢价可达20-30% |
| 流量被平台绑架，获客成本高 | 蓝海渠道竞争小，增长空间大 |
| 合规要求越来越严，风险增大 | 特定地区政策友好，营商环境优 |
| 高端人才被大厂抢走，招聘难 | 部分地区人才洼地，薪资成本相对较低 |

**避风港模型**：不是「出国就好」，而是「出去找风停的地方」。

### 出海四阶段
| 阶段 | 特征 | 用工模式 |
|------|------|---------|
| 起步期（出口） | 侧重贸易能力，依托代理商，无需当地文化 | 少量外派 |
| 开拓期（本地化） | 当地产能和运营能力，直面当地法规，需与当地文化交融 | 外派+Local首岗 |
| 腾飞期 | 整体经营体系化 | 体系化Local招聘 |
| 全球化 | 全球运营 | 全球化人才配置 |

## 二、市场选择方法论

### 四象限市场筛选
| 象限 | 特征 | 策略 |
|------|------|------|
| 蓝海市场 | 需求强+竞争弱 | 优先进入 |
| 红海市场 | 需求强+竞争强 | 差异化突破 |
| 坑货市场 | 需求弱+竞争强 | 坚决避开 |
| 无人区 | 需求弱+竞争弱 | 谨慎验证需求 |

**方法论来源**：BCG矩阵改良版

### 市场选择决策框架
- **看钱景**：哪里能赚钱就去哪
- **看产品**：你的东西在哪好卖
- **看用户**：你的客户在哪就去哪

**案例**：智能电器→欧洲；基因检测→东南亚-中东-欧洲；高端脑科学成像→美日-欧洲

### 六大出海环节
产品→营销→运营→客服→物流→合规，每环节均含本土化要素

## 三、出海HR核心痛点

### 两大大核心痛点
1. **人招不到**：渠道匮乏、信息差大
2. **招到了管不好**：管理复杂、文化冲突

### 底层关键因素
- **跨文化差异**：语言不通、思维方式不同、工作习惯差异
- **合规风险**：各国法规、税务、社保差异大，违规面临巨额罚款

## 四、人才构成与用工策略

### 团队配置原则
- **一号位**：前期应外派，同时培养一个Local地一号位
- **销售一定要Local化**（例如：欧洲一定要招白人）
- **HR一号位**：当地人，华人最好
- **HR团队**：需要有当地人

### 海外人效ROI
- 海外人效至少要做到国内的**2.5-3倍**
- 随着中企出海发展，海外跨国员工流动越来越多
- 在各个国家Local招聘当地员工

### 用工模式选择
| 场景 | 推荐模式 |
|------|---------|
| 出海初期未设主体 | EOR（省时省力，合规咨询支持，货币预算对冲） |
| 有主体、规模化团队、有当地HR | Payroll/HRSSC或Self-Support |
| 员工归属感要求高、规模大 | Self-Support（体系化但搭建初期艰难） |
| 完善公司HR政策 | BPO（通过Payroll外包完善HR政策） |

### 外派人员规划
- **<3-6个月短期出差**：商务签证、差旅政策
- **6个月以上长期外派**：劳动关系、薪资福利设计、定岗定责、工作签证、合规管理

## 五、外派薪酬体系

### 模式一：派出国平衡法（初期适用）
以员工在国内薪酬为基础，增加额外激励和津贴，确保生活水平不降低。
- 外派期间薪资具有连续性
- 匹配当地合规要求和生活标准
- 一般适合3年内的外派

**三均衡模型**：薪酬均衡×福利均衡×职业发展均衡

### 模式二：Local+薪酬（体系化阶段）
基于当地的薪酬体系，附加额外福利或津贴，实现本地化融合。
- 激励长期发展，解决后顾之忧
- 对价补偿个人与家庭为此的付出
- 有助于长期稳定性
- 一般适合3-5年内外派

### 外派补贴体系
| 补贴类型 | 标准 | 支付方式 |
|---------|------|---------|
| 生活补贴 | 满足当地中上水平住宅租赁 | 按月 |
| 艰苦补贴 | 根据派驻国环境、治安等分级，0-100USD/天 | 按天 |
| 住房补贴 | 职级不同金额不同，占基本工资20-50% | 按月 |
| 外派补贴 | 各个国家标准不同 | 按月 |
| 家属随行补贴 | 员工的1/2 | 按月 |
| 子女教育补贴 | 额外福利 | 按需 |
| 探亲假 | 额外带薪年假+探亲交通补贴（2次/年） | 按年 |

## 六、Local招聘与四阶段

### 出海四阶段中的Local招聘
| 阶段 | Local招聘重点 |
|------|--------------|
| 起步期 | 外派为主 |
| 开拓期 | 开始考虑国家经理岗，启动本地化 |
| 腾飞期 | 本地高级管理者，体系化合规化 |
| 全球化 | 新拓市场+关键职能，雇主品牌深化 |

### "首岗"人才画像
"能独当一面、懂中懂外、能开能管"的"六边形战士"
= 当地人脉资源 + 专业经验 + 跨文化沟通能力 + 对中方业务逻辑的理解

**现实**：几乎不存在！

### 本地化三大差异（卡点）
| 差异类型 | 内容 |
|---------|------|
| 信息差 | 核心问题是信息壁垒 |
| 认知差 | （跨）文化差异、世界观/就业观、政治经济环境 |
| 物理差 | 时区、地域、沟通、管理 |

## 七、全球HR管理体系

### 三大管理维度
1. **合规管控**：薪税社保合规、人才招聘合规、雇佣合规
2. **集团HR体系**：绩效、薪酬数据报告、跨文化
3. **合规管理体系**：制度合规、文件合规、劳动法合规

### 核心模块
- 全球商保、非试用期解除合规
- 薪税社保合规（发薪、社保、个税）
- 劳动合同、试用期、绩效、转岗、改进、解雇
- 反歧视与DEI、跨文化雇主品牌
- 保密与竞业禁止、数据跨境规划

### 三级闭环管控框架
**总部全球HR管控中心 → 区域HR共享中心 → 属地国家HR团队**

### 九大核心管控原则
合规优先·总部统管基线·三级协同·全周期覆盖·风险前置·降本增效·数据先行·属地优化·行为合规

## 八、成功公式

**出海人力成功 = 认知 × 合规 × 文化 × 优化**

| 要素 | 内涵 |
|------|------|
| 认知 | 打破国内思维，建立"合规+属地化"核心认知 |
| 合规 | 守住用工合规红线，避免天价罚款 |
| 选型 | 选对用工模式，降低成本与风险 |
| 薪酬 | 设计合规+激励的薪酬方案，留住核心人才 |

**四大要素缺一不可，通过七步走策略构建可持续发展体系**：
招聘→管理→复盘（低成本渠道招高适配人才 / 化解跨文化冲突 / 数据复盘持续优化）

## 九、AI在出海HR中的应用

### AI辅助海外招聘
- **极速筛选**：10分钟内完成1000份简历精准筛选
- **智能优势标注**：自动识别候选人核心竞争力，精准标注匹配度
- **自动生成报告**：一键生成结构化候选人报告

### AI生成薪酬和用工报告
- **极速生成报告**：1分钟生成某国家详细薪酬分析报告
- **定制最优方案**：根据企业需求定制发薪方案
- **合规与成本优化**：内置最新法规数据库，确保合规

### AI发展四代演进（锴哥判断）
| 代际 | 特征 | 代表 |
|------|------|------|
| 上一代 | 拖拽编排/纯聊/想什么做什么 | 扣子/GPT/Codex |
| 这一代 | 说一句真的去干 | OpenClaw小龙虾/Hermes |

## 十、用友薪福社Global CHO（龙虾）智能体体系

### 五阶段搭建路径
1. **基础启动**：配置运行环境、飞书接入
2. **搭建Wiki**：配置插件、知识库整理
3. **创建协作Agent**：法规检索/方案撰写/表哥/合规提醒
4. **设置个人偏好**：身份、输出风格、记忆清理
5. **开启自动能力**：文本优化、飞书日程绑定、自动合规扫描

### Agent体系
- 法规检索Agent：找法规、查矛盾
- 方案撰写Agent：写出海方案、做PPT
- 表哥Agent：数据统计分析、写汇报
- 合规提醒Agent：提醒法规更新、过时内容、合规审计
