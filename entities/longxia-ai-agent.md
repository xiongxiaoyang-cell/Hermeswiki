# 龙虾AI智能体项目

以"AI龙虾"为载体的出海人力资源管理智能体平台。

## 项目定位

**出海人力资源管理智能体平台**

- 初期核心Skill：国别用工指南、薪资查询、人力成本核算、简历预览
- 长期目标：覆盖人力资源"选用预留"全流程
- 核心价值：降低C端用户养龙虾门槛付费换时间和经验，为出海人力业务引流

## 商业模式：C→C闭环

| 要素 | 说明 |
|------|------|
| 用户分层 | 已熟练使用者 / 尝试效果不佳者 / 有意愿未行动者 |
| 核心机制 | 从C端用户萃取经验反哺其他C端用户，形成经验闭环 |
| 分成机制 | 用户既是内容贡献者也是销售者，按技能贡献比例获得分成 |
| 盈利核心 | Token消耗分润 + 分层产品收费，非直接售卖软件/产品 |

## 技术路径

- 统一入口（APP/聊天窗口），路由切换合作厂商
- 保障数据回收与Skill蒸馏
- 初期与360安全龙虾合作（垂直厂商，避免大厂复制风险）
