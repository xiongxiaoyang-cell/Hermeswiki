### 2026年4月27日
🔴**🏦社保与雇主税费** — [[raw/sources/03_区域政策/政策日报/20260427A|20260427A]]
- **法规**：Code on Wages, 2019 / Code on Social Security, 2020
- **变化**：基本工资须≥总CTC的50%（自4月1日生效）；ESIC全面推广至全印度；Gratuity增加约66%（基本工资从30%→50%CTC）
- **影响**：EPF和Gratuity成本预计上升5-15%；员工实得工资可能减少
- **信源**：P1 [✓] [Mint India](https://www.livemint.com/money/personal-finance/new-salary-rules-in-india-what-the-50-wage-norm-means-for-epf-gratuity-and-take-home-pay-11775526270823.html) (2026-04-27)

## ⚠️ 重大合规：2026年4月1日劳动法典全面实施

新法典整合4部旧法（Industrial Relations Code / Code on Social Security / Occupational Safety Code / Working Code）

### 关键变化
- 50%工资规则 → PF/Gratuity+5-15%雇主成本上升
- 各邦最低工资上调（UP邦+21%）
- 解雇保护扩大至更多企业

### 社保变化
- EPF：雇员12%+雇主12%（维持）
- Gratuity：10-20天工资/年

## 劳动法要点
- 各邦最低工资差异大（₹178-₹1,035/天）
- 工会力量强（尤其制造业）
- 外国人工作许可：L Card / Employment Visa
